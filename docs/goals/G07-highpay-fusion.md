# G07: 高配当検知×新シグナル合成（事前登録セルのみ）

## 目的
ユーザーの最終目標は「高配当の的中×ROI100%」。既存検証で:
- 万車券**検知**は有効（top3_sum AUC0.73・Q1帯で万車券率52%・doc14/doc21）
- だが**価格化（買って勝つ）**は公開オッズ内では全滅（doc21: 36セル全滅・
  「的中10-20%と万車券中心は市場価格上両立しない」）

→ 残る可能性は「**検知（有効）× 公開オッズ外の新シグナル（G04 money-flow / G06 風）**」の
合成のみ。これを事前登録セルだけで検証する。
