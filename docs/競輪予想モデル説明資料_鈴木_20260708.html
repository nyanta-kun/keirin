2段階ランクシステム（SS / S）
対象は7車立てのみ・三連複。ランクは発走15分前のリアルタイムオッズで確定
ランク確定条件（発走15分前）買い目狙い
SSガミ目（<7倍）カット後の有効目が 1〜3目三連複 1〜3点（各100円）点数集中・高効率
S有効目4目以上 ∧ gap12≥0.10三連複 有効目全点（ガミ目除外）確信度高・面で回収
  • 朝は gap12≥0.07 の7車レースを候補として通知し、15分前判定で SS/S/見送りが確定する2段構え。
  • Aランクは廃止（2026-06-28）: 低確信帯 gap12[0.07,0.10) は期待値が薄く購入対象外に。
  • 7車立て限定（2026-07-03）: 9車はS束ROI 94.8%（12ヶ月中6ヶ月赤字）で除外。7車はROI 149.1%。
購入フィルタ（3条件・発走15分前に判定）
「当てても損」「低配当構造」「軸の確信不足」を事前に除外
フィルタ条件検証根拠（最終オッズ上限値）
① ガミ回避全購入目 三連複 ≥ 7.0倍min≥7帯がプロフィットエンジン: ROI 2.18 [CI 1.85-2.51]。見送り帯[3,7)はROI 0.80で回収不能（14,892R検証）
② 合成オッズ有効目の合成オッズ ≥ 8.0低配当構造レースの除外。6月検証でフィルタ全体のROI +11.6pt
③ gap23モデル2位-3位の確率差 ≥ 1.0pt軸2頭の確信度担保。②と合わせ ROI 130.2%→141.8%（対象-13%・除外分は63.3%）
  • ガミ閾値は検証を経て 5.0 → 7.0 に引き上げ（2026-07-01）。閾値を下げるほどROIは単調悪化（≥6→1.75、≥5→1.40、≥3→1.05）。
  • 不採用と結論した案: ガミ緩和（最低3倍∧合成10倍は数学的に空集合）／見送りレースのオッズ復活買い（全変種ROI<1）。
