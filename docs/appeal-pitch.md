# 穴車AI — 競輪AI予想システム アピール資料

> 作成: 2026-06-03

---

## 1. コンセプト

**「AIが発見する、市場の盲点」**

競輪の3連単・3連複予想に機械学習を適用し、
オッズ市場が読みにくい "高配当レース" を自動で特定する予想システムです。

---

## 2. なぜ競輪 × AI なのか

| 理由 | 詳細 |
|------|------|
| **豊富な数値データ** | 競走得点・勝率・ギヤ比など選手データが公式公開 |
| **高頻度開催** | 全国9〜12会場 / 1日250レース超のデータが毎日蓄積 |
| **払戻構造の優位性** | 6車立てレースは組み合わせ数が少なく期待払戻が高い |
| **市場の非効率** | 人気 = 低配当が崩れる "読みにくいレース" が存在する |

---

## 3. 予想エンジン

- **モデル**: LightGBM（勾配ブースティング機械学習）
- **特徴量**: 24項目（選手の競走得点・場別勝率・6ヶ月勝率・脚質・ギヤ比・地元フラグ 等）
- **学習データ**: 約95,000レース（3.5年分 / 2023年〜）
- **対象レース**: 6車立て以下のみ（払戻が最も有利な条件を選別）

---

## 4. 3段階ランクシステム

AIスコア（確率ギャップ × 市場読み指標）で 3ランクに分類し、買い目を最適化します。

```
【SSランク】 〈市場の盲点レース / 3連単〉
  AIが1着・2着の確率差を掴んでいるのに、市場が混戦と判断しているレース。
  1日 0〜1件の超絶選別。高配当を狙う。

【Sランク】 〈AIの確信度が高いレース / 3連複〉
  AIが1位・2位を高信頼度で特定。3連複で安定した収益を目指す。
  1日 3〜4件。

【Aランク】 〈バランス型 / 3連複〉
  中程度の確信度。3連複で中オッズを狙う。
  1日 2〜3件。
```

**投資規模の目安**: 全ランク合計 1日 約2,400円（1件300円 × 8件）

---

## 5. SSランクの仕組み

> **「AIは順位を見えているが、市場は混戦と見ている」** ——これが高配当の源泉

- **条件①**: AI予測の1位・2位確率差が大きい（= AIが着順を確信）
- **条件②**: 市場期待値に対して1位確率が低い（= 市場は混戦扱い）
- **結果**: 市場オッズが高いまま → 的中時の払戻が大きい

5車立て・6車立ての小規模レースに多く出現。3連単 3点（300円）で購入。
