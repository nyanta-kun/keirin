# ≤6車: 指数下位（モデルランク4位以下）の好走条件 — 網羅的残差分析（2026-06-13）

> 問い: モデルがレース内 pred_prob ランキングで4位以下に置いた選手が top3 に入る（好走する）
> 条件は何か。doc13 で「指数下位の単騎逃げ」の過小評価（残差 TR+7.1%/TE+27.6%）が1件
> 見つかっており、その一般化として50セルを1次元走査した。
> スクリプト = `scripts/exp_lowrank_goodrun_wt.py`（--from/--to対応）
> model=lgbm_wt_eval（2023-07〜2026-02学習・2026-03以降OOS。週次再学習lgbm_wtは不使用）
> TR 2024-01〜2026-02（in-train＝残差は保守側）11,658人 / TE 2026-03〜2026-06-12（OOS）1,478人
> doc18セマンティクス準拠: ≤6車=出走表基準・ランキング=全エントリー（欠車含む）・top3=`between(1,3)`
> 市場示唆P(top3)=trio**最終オッズ**盤面逆算（q∝Σ1/odds・×3正規化）＝確定直前の市場評価。
> **本分析は記述的残差分析であり「買える」の主張は一切しない**（ROI検証は3期間プロトコル＋live実測が別途必要）。

## 結論（要約）

1. **判定A（モデル・市場とも過小評価）= 50セル中 0件**。指数下位の好走を「モデルも市場も
   見落とす」形で系統的に説明する条件は見つからなかった。期待偽陽性 ≈0.6セルに対し実測0＝
   doc18 の「公開オッズ内の情報にエッジ無し」と整合する陰性結果。
2. **判定B（モデルは過小評価・市場は織込済）= 5件**。最大は「**市場がモデルより2+上に評価
   /市場はtop3級に評価**」セル（モデル残差 TR+9.0〜+11.1% / TE+13.1〜+17.7%）——モデルが
   下位に置いた選手を市場が高く買っているときは実際に好走するが、**市場示唆（55〜57%）が
   既に実績を言い当てている**＝モデルだけが持たない情報（並び・直前情報・人気）を市場が保有。
   fav_mismatch（doc13）と同じ構造の較正ギャップでありエッジの証明ではない。
   他は「単騎」「top3_sum Q1/Q2（波乱形状）」＝既知レバーの裏面。
3. **doc13 P4-3（拮抗×逃げ1人×rank4+）の残差は正確に再現**（TR +7.1%・n=143。doc13の
   +7.1%/n=244 と同値）。TE も +19.5% と同方向。ただし **Wilson 95%CI では TR 下限38.2% <
   予測39.1% で有意に届かず、TE n=13 と運用不能規模**＝判定C(n)。市場残差も +5.4/+16.8% と
   正で「市場も完全には織り込んでいない」可能性は残るが、n 蓄積速度（OOS 3.5ヶ月で13人）から
   追跡対象にしない。doc13 の is_lone_nige 不採用の結論は変わらない。
4. **較正の全体像: rank4+ の pred_prob は top3 確率としてほぼ較正済**（TR 全帯で残差±2pp内・
   isotonic はほぼ no-op）。副所見として、**市場示唆はrank下位ほど過大評価**（rank6: 市場20.3%
   vs 実績17.9%＝-2.3pp、TR/TE同方向）＝古典的 favorite-longshot バイアスがtrio盤面にも存在し、
   **下位ランクについてはモデルの方が市場より較正が良い**。ただしこれは「下位を買うと損」方向の
   バイアスであり、エッジ（過小評価）ではない。

## 設計（事前指定）

- 対象 = ≤6車（出走表基準）・n≥4・結果確定レースの**モデルランク4位以下**の選手。
- 残差2系統: residual_model = 実top3率 − mean(isotonic較正済pred) / residual_market = 実top3率 − mean(市場示唆P(top3))。
  isotonic は TR rank4+（市場カバレッジあり主標本）で fit。市場カバレッジは TR/TE とも100%・欠車率0.9%。
- 判定（事前固定）: **A** = TRで実績Wilson CI下限がモデル・市場双方の平均予測を上回り、TEで両残差とも同方向
  / **B** = モデル側のみ成立（市場は織込済） / **C** = ノイズ（CI跨ぎ・TR/TE不一致・n不足 TR<100 or TE<15）。
- 走査次元（1次元のみ・組合せ爆発させない。例外=doc13再現の2セル）: ランク(4/5/6)・脚質(逃/両/追)・
  ライン役割(単騎/先頭/番手/3番手+)・ライン規模・級班/期/ギアのレース内相対・バンク(周長/ドーム/直線/カント)・
  車立て・グレード・開催日次・top3_sum四分位・gap12四分位・市場との不一致 = **計50セル**。
- 多重比較への防衛: TR有意（一側α≈2.5%）×TE同方向（×0.5）の期待偽陽性 ≈ **0.6セル**。
  A判定はさらに市場側のTR有意が必要なため期待値はこれ未満。

読み方:
- 「市場不一致」2セルは fav_mismatch（doc13実験2）の選手版＝**モデルと市場の意見が割れたとき、
  下位側の真値は市場寄り**。市場示唆が実績をほぼ言い当てている（TRでは市場の方が過大なくらい）

- 「単騎」は市場側がむしろ過大評価（-1.7/-3.0pp）＝単騎は「市場が知ってる以上に来ない」。

- **残差 +7.1% は doc13 と同値で再現**（期間・ランキング規約[doc18修正版]が違っても安定）。
  方向も TR/TE 一致・市場残差も両期間正＝唯一「A候補だった」セル。

  狭い文脈のみ。doc13 の is_lone_nige 特徴不採用（AUC不変・残差解消せず）の結論は不変。

## Part4: 判定C の付記（方向だけ出た近傍セル・追わない）

- 脚質「逃」: TE +6.3% だが TR +0.8% で非有意＝C。ドーム: TE +6.9% / TR -1.2% 方向不一致＝C。
- top3_sum Q4(堅い): 残差 **-5.1/-5.2%**（TR/TE同方向の**過大**評価）＝堅いレースの下位は
  モデル予測ほども来ない。市場も同様に過大（-3.8/-5.0%）＝「堅い形状の下位は買うだけ損」という
  既知の負方向の再確認。
- rank6: 市場が +2.3pp 過大評価（TR/TE同方向）でモデルの方が較正良＝longshot バイアスの選手版。

## 判定と次のアクション

- **クローズ**。50セル走査で A=0／B=5（全て市場既知 or 既知レバーの裏面）／C=45。
  「指数下位の好走」は (1)市場が知っている（並び・直前情報） (2)波乱形状（top3_sum で捕捉済）
  (3)P4-3（実在するが微小・n不足・特徴化失敗済）で説明が尽きる。
- 変種（2次元組合せ・帯の切り直し）は多重比較の罠のため追わない（18軸・doc16の教訓）。
- 残る前進路は本分析の範囲外＝公開オッズを超える情報源（money-flow G04 / 風は不通過済）と
  live実測（picks_history）のみ（docs/goals/README.md の路線通り）。
- P4-3 は「既知の較正破れ」として記録に留める（live で5車・拮抗×逃1人が数百R溜まる見込みは無い）。
