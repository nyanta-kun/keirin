## Sランク＝三連単1着固定フォーメーション（2026-07-10 採用・両立実装）

三連複の残り物からのS構成（下記・見送り）とは別に、**券種を変えた S** を採用:

```
買い目: 1着=指数1位固定 / 2着=指数2,3位 / 3着=全通り（2×(n-2)点）
S:  gap12>=0.15 ∧ 購入全目の三連単オッズ min>=10 → 100円/点  (7PLUS_ST)
S+: さらに gap12>=0.25 ∧ gap34>=0.04 → 200円/点 増額        (7PLUS_STP)
```

| 構成 | R/日 | 的中率 | 真OOSプール(80日) | 2025通年 |
|---|---|---|---|---|
| S (0.15×min10) | 9.9〜10.2 | 18.4〜19.5% | **115.1%** | 113.5% |
| S+帯 (0.25×g34 0.04×min10) | 1.8〜2.1 | 22.6〜24.3% | **150.1%** | 131.3% |
| 両立(S+増額込み) | 10.2 | 19.5% | **121.1%** | — |

- 効くレバー: gap12（1位の1着率）と **全目min≥10のレース単位ガミ条件**（順当決着の安配当レース除外）。
  ガミ条件なしは的中38%でもROI 84%で不成立。
- 買い目=全通りは SS と同じ理屈（3着は誰でも良い）。的中条件 = 1着=指数1位 ∧ 2着∈指数2,3位。
- 注意: 2025年1-10月はインサンプル。単月赤字あり（S構成で2025年中5/12ヶ月・12月OOS 89.3%）。
- SS(三連複)と同一レースで併存購入する。decisions キーは `{rk}#ST`・picks_history は `{rk}#7ST`。
- ハーネス: `scripts/exp_trifecta_1st_fixed_wt.py`

## Sランク（三連複の残り物から）検討（2026-07-10）→ 見送り

「的中率20%まで許容して対象を増やし、指数下位の買い目を削ってROI確保」を
64構成スイープ（k∈{2,3,4,all} × gami∈{4,5,6,7} × gap12∈{0.07,0.10} × gap23∈{0,1pt}、
S候補=SS落ちレース、2025年18,322候補R + 2026-06）で検証した結論:

1. **gami<7 に緩和した構成は全滅**: 2025年（大半インサンプル=楽観上限）ですら ROI 76〜90%。
   SSの利益源は「全目min≥7」そのもので、これを崩すと期待値が消える。
2. **買い目削り（指数下位カット）は逆効果**: 全構成で k=2/3/4 < all目。
   払戻は指数下位（人気薄）の目に集中しており、削ると的中率と平均払戻が同時に落ちる。
3. gami7 を維持して gap12/gap23 だけ緩和した S-A/S-B はインサンプル 107〜115% だが
   **真OOSで崩壊**（2025-11: ROI 11〜14%・的中6-9% / 2025-12: 64〜86%）。
   2026-06 の好成績（155〜186%）は n=21〜27 の小標本。

| S候補構成 | 2025計 | 2025-11 OOS | 2025-12 OOS | 2026-06 OOS |
|---|---|---|---|---|
| S-A: gami7×gap12,gap23両緩和 (1.4R/日) | 107.5% (24.5%) | **13.7%** | 64.5% | 155.4% |
| S-B: gami7×gap12のみ緩和 (0.8R/日) | 115.2% (23.7%) | **11.2%** | 85.7% | 186.2% |

**結論: SS落ちプールに再現性のあるエッジは無い。S新設は見送り、SS一本で運用。**
対象を増やす場合の唯一の候補は S-B を「参考表示（購入なし）」で live 追跡し実勢確認後に昇格。

- ハーネス: `scripts/exp_s_tier_wt.py`

## 背景

「gami閾値7倍だと対象レースが減るので6倍に緩和したい」という要望から調査。

## 重要発見: gami閾値は 2 つの異なるセマンティクスが混在していた

| セマンティクス | 導入 | 挙動 |
|---|---|---|
| **レース単位除外** (doc48) | 2026-06-16 | min(全目) < 閾値 → レースごと見送り。全目購入 |
| **買い目カット** (doc49〜) | 2026-06-28 | 閾値未満の目をカット。残 1-3 目=SS / 4目以上=S |

カットセマンティクスでは **閾値を下げると購入は激減する**:
- 閾値を下げる → 低オッズ目が買い目に残る → SO≥8 フィルタで全滅
- 実15分前オッズ (7/9) 再現: thr 7→6 で購入 4件→**0件**
- gami4×SO5 も同様に減る（6月OOS 186R→107R）: gami が下がると SS(カット後≤3目) が成立しなくなる

## 最終決定 (2026-07-10): SS/S を廃止し R ランクに一本化

**レース単位セマンティクス（doc48 回帰）に全面移行**:

```
min(全目オッズ) >= GAMI_THRESHOLD(7.0) ∧ gap12 >= 0.10 ∧ gap23 >= 1pt
→ 7PLUS_R: 全目購入（カットなし・SOフィルタ廃止）
```

- SO≥8 は全目合成だと構造的に通らない（5目で合成≈1.5）ため廃止
- 的中条件 = 「AI 1位・2位の両方が3着内」→ **的中はオッズ非依存（モデル起因）で頑健**
- 払戻は確定オッズで支払われるため、バックテストの払戻側は live 忠実
  （オッズ時点差の影響は min≥7 ゲートの所属のみ）

### 選定根拠: 2025年 月別（lgbm_wt_train_only・11-12月=真OOS）

| 指標 | 値 |
|---|---|
| 2025通年 | 1,465R (4.0R/日)・**的中率 29.3%**・**ROI 147.6%** |
| 月別ROI | 13ヶ月中12ヶ月が100%超（最低 2025-08 の 99.7%） |
| 真OOS | 2025-11: 27.9%/120.1%・2025-12: 31.5%/139.6%・2026-06: 39.4%/299.1% |

ユーザー要件「的中率30%×ROI100%」を満たす。旧SS（カット方式）は的中率9-13%で
7月live赤字だったため置き換え（S は min≥7∧SO≥8 で R と同一買い目 → R に自然吸収）。

## 併せて検証・見送りにした事項

### 朝時点の人気偏り除外（オッズベース）→ 見送り

朝オッズは直前の購入不成立をほぼ予測しない（朝全目<7でも23%が直前復活・朝SO<8でも15%復活）。

### 構造統計（得点/指数の偏り）による朝除外 → live 蓄積待ち

score_sd / score_gap2r / pred_sd / pred_top2sum（7/7 から decisions に記録済み）:
- オッズ落ち予測としては**逆相関**（偏り大→thirdsが人気薄→3連複オッズ高→むしろ成立しやすい）
- **購入の質**とは強相関: pred_top2sum 上位20%帯の購入 ROI 63.6% vs 下位20%帯 463.7%
  → R ランクの質フィルタ候補として live 蓄積後に昇格判断
