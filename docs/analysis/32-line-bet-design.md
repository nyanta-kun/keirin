# ライン構造ベース買い目設計検証（2026-06-15）

> **Phase2 不通過（全3戦略・全期間 ROI < 100%）**
> 結論: 買い方変更（軸シフト）もROI改善に寄与しない。市場効率の壁はモデル精度だけでなく買い目設計にも適用される。

## 仮説

doc31（ライン先頭強度特徴量）では「AUC↑・ROI↓」だった。これは「特徴量が市場に既に織り込まれており、的中率が上がった分オッズが下がる」ためと解釈した。

**別仮説**: 特徴量を追加するのではなく、ライン情報を使って **軸選択を変える** ことで、市場が誤評価している選手を軸にできないか？

具体的に: 指数差（gap12）が大きい場合は現行戦略（指数通りの軸）、指数差が小さい場合はライン構造で軸を決める 2×2 フレームワーク。

## 検証設計

- TRAIN 2023-07-01〜2025-06-30 期間限定学習（リーク無しモデル）
- VAL 2025-07-01〜2026-02-28 / HOLD 2026-02-28〜2026-06-12
- 三連複2軸流し（pred1+pred2 → thirds・ガミ≥5倍足切り）
- パラメータ: 強番手閾値=5.0点 / 拮抗閾値 gap12<0.06

## 解釈

### gap12 小 = 拮抗レースは「最もROIが低い帯」

仮説「拮抗レースでライン情報を使えば逆転を狙える」の逆が起きている:

- gap12 < 0.03（最拮抗）: S1 が 0.0%（全不的中）
- gap12 0.03-0.06（準拮抗）: 74%程度
- gap12 0.06-0.10（やや差あり）: 両戦略で >100%（ただし少数）

**拮抗レースは「モデルも市場も迷っている」状態** = ランダム性が高くライン情報でも突破できない。
gap12 0.06-0.10 の >100% は、指数差がある程度存在し「確度が高い軸」が決まるが、まだ波乱の余地がある帯の特性（top3_sum Q1 の別表現）。

### 買い方変更は市場効率の壁を超えられない

- **特徴量追加（doc31）**: AUC↑ ≠ ROI↑ → 市場が同じ情報を既に価格に反映
- **軸シフト（本検証）**: ライン構造で選んだ軸も市場が既に適切な評価 → ROI↑ならず

軸を変えても「市場が誤評価していない選手を軸にするだけ」。公開情報の範囲内ではエッジは得られない。

## 結論

- **新戦略は全て不採用**（買い目設計変更なし）
- 2×2 フレームワーク（gap12大/小 × ライン強弱）は統計的支持なし
