# ≤6車へのライン力学転用 — 結論: 力学は再現するが現行戦略が上回る＝現行維持

> 2026-06-09 / `scripts/exp_le6_line_dynamics_wt.py` / pooled lgbm_wt・≤6車・三連複 / TRAIN 5,045R(2023-07〜2026-02)・TEST 479R(2026-03〜) / 払戻=wt_odds最終(上限値)

## 背景
7+検証の副産物「本命(逃)ラインは飛ぶ時に番手も道連れで崩れ、別ライン1-2が台頭」(`docs/analysis/05` 追検証9-10)を、**収益実体のある≤6車**に転用できるか検証（運用モデル・買い目は変更しない・検証のみ）。

## Part A 精度: ライン力学は≤6車でも再現する
- pred1位バスト率(3着外): **16.4%**（7+の20%より低い＝小フィールドは本命が堅い）
- pred2位が本命と同ライン: 59.5%
- **バスト時 top3率: 別ライン最上位RLH 84.6% > pred2位 77.1%**（7+と同方向）
- バスト時 非r1選手top3率: **本命と同ライン48.3% / 別ライン68.5%**（道連れ崩壊・別ライン台頭が≤6車でも成立）

→ ライン力学は≤6車でも構造として再現する。

## Part B ROI: だが現行2軸流しがライン版を上回る

| 選別 | 現行2軸流し(pred1,pred2+pred3-5) | ライン版(別ライン頭軸/同ライン差替) |
|---|---|---|
| ALL | **89/87%** | 80/81% |
| 開(top3_sum<中央) | 96/99% | 87/104% |
| **Q1相当(top3_sum≤p25・収益ゾーン)** | **111/106%** | 99/98% |

- **ライン版は現行を上回らない**。特に収益の核Q1で現行111% vs ライン版99%(TRAIN)。
- ライン版は的中率を落とす(Q1で40.5%→34.2%)＝別ライン頭への差替が逆効果。

## 解釈・結論
- ≤6車は①本命バストが少なく(16%)②pred2位が同ラインでも十分強い③ライン情報は**既にFEATURE_COLS_WT(line_group/n_senko等)経由でモデルが取り込み済**。よって明示的なライン軸置換は的中率を落とすだけで+EVに繋がらない。
- = **現行≤6車戦略(2軸流しpred1,pred2)は正しく、変更不要**。ライン力学は「実在するが、モデルが既に内包しており、明示ルール化の追加価値なし」。
- 7+でライン力学が稼げないのは市場効率、≤6車で明示ルール化が不要なのはモデル内包＝**どちらもライン力学を別途の買い目ルールにする意味はない**ことで一貫。
