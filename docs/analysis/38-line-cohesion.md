# doc38: ライン連携コヒージョン特徴量実験（2026-06-15）

> **Phase1 不通過（AUC +0.0001）・採用見送り**

## 仮説

競輪のライン（チーム走行）は毎レース異なる組み合わせ。
特定リーダー L と番手 F の過去共走 top3 率は
個人成績では捉えられない「ライン相性」シグナルを持つ可能性がある。

## 新特徴量

| 特徴量 | 内容 |
|---|---|
| `partner_top3_rate` | リーダー L と過去共走した際の F の top3 率（expanding mean, shift=1） |
| `leader_win_rate` | F が過去に追走したリーダー L の勝率（同上） |

### データ特性

- `partner_top3_rate`: 非ゼロ率 17.6%（大半が初回共走 = 0 埋め）
- `leader_win_rate`: 非ゼロ率 6.9%（極めて疎）
- 上位 12 特徴量に入らず

## 解釈

**なぜ効果がないか**:
1. **データ疎性**: 特定ペアの共走回数が少なく（平均 5〜10 回程度）、統計的に信頼性が低い
2. **既存特徴量の代替**: `line_frac`（ライン支配力）・`is_line_leader` がライン強度を既に捉えている
3. **ライン相性の安定性**: 競輪のライン組みは大会ごとに変わり、長期 H2H の履歴が短期では機能しにくい
