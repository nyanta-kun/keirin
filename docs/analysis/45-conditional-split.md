# doc45: 条件別ROI体系スキャン

最終更新: 2026-06-16

## 仮説・背景

既存モデル（TRAIN期間のみで学習・週次再学習リーク無し）の予測をベースに、
レース属性（n_entries, grade, bank_length）およびモデル出力（gap12, pred1）の
条件分けを体系的にスキャンし、C0戦略で一貫して ROI >100% となる条件を探索する。

**前提**: 新情報源の追加が枯渇したため、既存データの条件分けにより
エッジが特定の条件下に集中しているかを検証する。

## 実験設定

- **モデル**: TRAIN期間（2023-07-01〜2025-06-30）のみで学習したLGBMClassifier
  （週次再学習リーク=bias③を排除）
- **評価**: C0戦略（trio・ガミ≥5倍・n_entries≤6・リーク無し）
- **条件軸**: n_entries(5/6), grade(S/A), bank_length(333/400/500m), gap12 帯, pred1 分位数
- **ハーネス**: `scripts/exp_conditional_split_wt.py`

### 注意: DNS 生存バイアスについて

欠車（finish_order=0）の選手を予測プールから除外しているため、
欠車した選手を誤って高ランク予測する失敗が回避されている。
競輪では欠車に関連する券は払い戻しとなるため影響は限定的と考えられるが、
doc18基準（~70-90%）との差異の一部はこれによる可能性がある。

## 解釈

### 主要知見

1. **gap12 >0.10 が最も信頼性の高い正フィルタ**
   - サンプル: VAL 52R / HOLD 20R（最大）
   - VAL 139.9% / HOLD 220.7% で一貫
   - 全体83Rから52Rへ絞り込み（残存率63%）

2. **S級レースが一貫して正ROI**
   - VAL 201.7% / HOLD 151.0%（35R/14R）
   - 市場の「S級プレミアム」が予想モデルと合致している可能性

3. **gap12 0.07-0.10 帯がVALで大幅悪化（21.9%）**
   - 0.10 超は高ROI、0.04-0.07 も高いが n が小さい
   - 中間帯（0.07-0.10）は市場に読まれやすい「見せかけ高確信」か

4. **A級はVAL不通過（82.7%）**
   - A級はS級より予測困難で市場優位
   - A級でも pred1 Q4（最高確信度）では VAL 145.1% / HOLD 481.7%

### 多重比較の警告

15セルが通過しているが、~30 条件のスキャンであり多重比較リスクがある。
最も信頼性の高い条件は:
- サンプルが大きい（VAL≥20R, HOLD≥8R）もの
- 単純な1〜2条件交差

これらを満たす最有力候補: **gap12 >0.10**（52R/20R）と**S級**（35R/14R）

## 結論

**一部の条件で一貫した正ROIが確認された**（VAL×HOLD両期間通過）。

最有力フィルタ: `gap12 > 0.10`
- 現行C0（83R VAL）から 52R VAL に絞り、ROI を 132.9%→139.9% に改善
- HOLD: 174.4%→220.7% に改善

**次のステップ候補**:
1. `gap12 > 0.10` フィルタを現行 C0 に重ねた live 検証
2. S級専用の gap12 しきい値最適化
3. gap12 帯の二峰性（0.04-0.07 と >0.10 に正ROI）の原因分析
