# gap13パターン×少点数3連単（2026-06-12・打ち切り記録）

> パターン: 指数1位−3位差≥20pt（≤6車の66%が該当・1位3着内85%/一方96%でOOS較正良好）。
> 構成: S1'=R1頭・2着=指数3,4位・3着=3位指数から15pt以内 (~4点) / S2'=+R2を3着からも外す (~2点)。
> 検証は全て doc18 セマンティクス（リーク無し期間限定学習・出走表基準・欠車void・最終オッズ上限値）。

## 結果

- **6車**: S1' 78/65/78%・S2' 78/46/94%（TR/VA/HO）→ 死。除外条件(top2Q4/同ライン/gap12大)は逆効果（的中も削る・doc12の的中↔オッズ1:1連動）。
- **5車のみ損益分岐近傍**: S1' 96/90/110%・S2' 104/85/177%(除最大100%)。doc17の「5車にエッジ濃い」と独立に一致。
  ただし TR489/VA115/HO65R・CI[26,375]級・VAL割れ → **棄却も採用もできないが、5車は~0.5R/日で運用不能 → 打ち切り**(ユーザー判断)。
- **7+**: S1' 80/74/69%・S2' 79/75/67%（TR35.7k/VA12.1k/HO5.3kR・CI全幅<100%）→ **大標本で確定的に死**。9車も同様(59-84%)。

## 再開条件
