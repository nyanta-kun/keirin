# #4 較正確率→Kellyステーク傾斜 — 検証結果（2026-06-09・DEFER）

> ロードマップ `docs/analysis/08-le6-roadmap.md` #4 の検証。
> スクリプト=`scripts/exp_kelly_stake_wt.py`（再利用すること）。
> 転用元: `exp_7plus_value_wt.py`(isotonic較正+Plackett-Luce)・`exp_stake_tilt_wt.py`(≤6車collect+top3_sum傾斜)・`roi_robustness_wt.py`(CI)。

## 問い
固定3点フラットでなく「較正勝率に比例した賭け金(=Kelly)」で**資金成長率 E[ln m]** を最適化できるか。
2仮説を分離検証:
- **H-A (Kelly-on-value)**: ≤6車に 較正P > 市場示唆P の真の誤価格があり、Kellyサイズで成長率↑。
- **H-B (Kelly-on-selection)**: 値の誤価格は無いが、top3_sum傾斜の連続版としてKellyが離散ティアを成長率で上回る。

## 設計
- スコープ: **S/A 層（三連複2軸流し3点）**。SS(三連単・N<100)は別券種のため除外。
- モデル `lgbm_wt_eval`（holdout・test≥2026-03 はOOS）。isotonic較正は**TRAINのみでfit**（test漏洩なし）。
- 同時確率=Plackett-Luce(条件付き)＋周辺積(独立近似)の2種。較正母集団=TRAIN全board trio。
- 成長率比較: 各手法は重みベクトル w_i のみ異なり、レバレッジ s* を TRAIN で最適化(Σln(1+s·X_r)最大化)→ TEST OOS で g=E[ln m_r] を bootstrap CI 付き評価。重み形状(均一/ティア/エッジ比例)の優劣だけを切り分ける。
- TR 3943R / TE 411R（≤6車 S/A）。払戻/オッズ=最終=**上限値**。

## 結果

### (1) 市場効率診断 — H-A 否定（7+ と同一構造）
PL較正・TEST(6,776 trio):
```
P_cal = +0.0144 + 0.773·P_market   corr=0.819
P_cal>P_market の「割安」組(3,693件): モデル主張 0.0706 / 実的中 0.0439 ≈ 市場示唆 0.0422
```
**較正後確率は市場価格をなぞるだけ**。「割安」と判定した組の実的中は市場示唆とほぼ一致＝**市場が正しく、較正確率に独立エッジ無し**。較正信頼性(十分位)は単調で健全（モデルは較正済みだが市場を超えない）。7+ value-bet決定実験(`docs/analysis/05`)と完全に同じ結論。

### (2) value-bet 対照の ★再現 は新エッジではない
較正P×odds≥ev_min の全trio等額買いは TRAIN/TEST とも 221〜291%(全ev_min ★再現)。だが(1)が示す通りこれは**本命選択＝既存システムの336%エッジの別表現**（低オッズ favorite を最終オッズ=上限値で買っている）。市場診断（実的中≈市場示唆）が独立エッジ不在を示すため、**較正value由来の新規収益とみなさない**。

### (3) 資金成長率比較 — 選別(top3_sum)が主役・Kelly単体は劣後
Plackett-Luce:
| 手法 | s* | TRAIN g | TEST g | TEST 95%CI | ROI(unit) | 購入R |
|---|---|---|---|---|---|---|
| flat | 0.048 | 0.049 | 0.092 | [+0.05,+0.13] | 279% | 411 |
| **tilt(top3_sum)** | 0.050 | 0.152 | **0.297** | [+0.18,+0.41] | 627% | 183 |
| kelly単体 | 0.743 | 0.167 | 0.212 | [+0.11,+0.32] | 1033% | 318 |
| **kelly×tilt** | 0.398 | 0.286 | **0.443** | [+0.26,+0.64] | 1599% | 156 |

周辺積(独立近似)では **kelly単体 g=0.048＝flat以下**（確率モデルの質に依存して脆い）。kelly×tilt も 0.268 で傾斜成分が牽引。

**読み取り:**
- flat が最弱＝何らかの傾斜は有効（既知）。
- **tilt単体(0.30) > kelly単体(0.21)**: top3_sum選別の方が成長レバーとして強い → **H-B 確定**。
- kelly単体は確率モデル依存で脆い（PL 0.21 / 周辺積 0.05）。較正の手間に見合わない。
- **kelly×tilt が点推定最大(0.44, ROI1599%)** だが CI が tilt と重複・**156R小標本**で統計的に未確立。

## 結論 / 判定 = DEFER（本番化しない・watch-list）
- **#4 の前提（較正Kellyで成長率を稼ぐ）は成立しない**。per-combo の value エッジは ≤6車でも無い（市場効率・7+と同型）。
- 成長レバーは **top3_sum 選別**（=方針A・既に `--stake-tilt`/`--upset-gate` で opt-in 実装済）。較正Kelly単体はこれに劣後し、確率近似に脆い。
- **kelly×tilt（波乱帯ゲート × エッジ比例サイズ）だけが tilt 単体を点推定で上回る可能性**を示すが、(a)CI重複で未確立 (b)N=156R (c)全て最終オッズ=上限値で、しかも傾斜が集中する高オッズ波乱帯は**朝→確定ドリフトの影響が最大**の帯。
- → **本番反映せず**。`top3_sum` 傾斜のライブ実測（ロードマップ #3）と朝オッズドリフト実測（#2前提）が先。kelly×tilt はそれらが揃った後に再評価する候補に格下げ。

## 副産物・次への接続
- isotonic較正＋PL同時確率の部品は ≤6車でも**較正済みである**ことを確認（十分位単調）。ガミ判定の精度補助には使えるが、ステーク傾斜の源泉にはならない。
- 残る論点は #4 でなく **#2(中間オッズ昇格)/#3(波乱ゲート本番化)** に集約。両者とも**朝オッズドリフト実測**が共通ゲート。
