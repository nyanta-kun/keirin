# 三連複 ライン軸2 + 指数下位除外 ROI 検証（2026-06-16）

> **主要結論**
> 1. **ライン軸2（同一ライン指数2位への変更）は逆効果** — 軸2が変わる64%のレースでROIが 92.4%→83.5% に悪化
> 2. **指数下位除外（R1/R2）も悪化** — VAL 62〜52%（現行73%を大きく下回る）
> 3. **AI確率順が最善** — AI 2位はモデルが全特徴量（指数含む）を統合した最良の2番手予測

## 検証の動機

ユーザー提案：
「２軸目に１軸目の同一ライン得点2位を入れ、指数一番下または２番目までは３列目から除外する」

## 戦略定義

| 戦略 | 軸2 | 流し相手 | 点数 |
|---|---|---|---|
| **S0** 現行 | AI確率2位 | AI確率3〜5位 | 3点 |
| **L1** ライン軸2+下位1除外 | 同一line_group内指数2位 | 残り全員 − 指数最下位1名 | 約2.9点 |
| **L2** ライン軸2+下位2除外 | 同一line_group内指数2位 | 残り全員 − 指数下位2名 | 約1.9点 |
| **R1** AI軸2固定+下位1除外 | AI確率2位（変えず） | AI3〜5位 − 指数最下位1名 | 約2.9点 |
| **R2** AI軸2固定+下位2除外 | AI確率2位（変えず） | AI3〜5位 − 指数下位2名 | 約1.9点 |

**フィルタ**: ≤6車 / ガミ(S0ベース3点最安)≥5倍  
**line_group**: wt_entries.line_group カラム（NULL率0%・全期間完全カバー）  
**フォールバック**: pivot1が単独ライン(line_size=1)の場合はAI確率2位を使用

**全戦略 Phase2 不通過（VAL≥100% AND HOLD≥100% 未達）。現行S0が最良。**

## 軸2変更の影響分析（VAL期間・76R）

| ケース | R数 | 割合 | S0 ROI | L1 ROI |
|---|---|---|---|---|
| 軸2が変わる（L1 ≠ S0） | 49R | **64%** | 92.4% | **83.5%** |
| 軸2が同じ（L1 = S0） | 27R | 36% | 38.5% | 38.5% |

**軸2が変わる64%のレースで L1 は S0 より 8.9ポイント悪化**。

## なぜライン軸2は効かないか

```
AI確率2位の強み:
  モデルは race_point・勝率・ライン位置・その他38特徴量を統合して
  「3着以内に入る確率」を予測している。
  AI2位 = 全情報を統合した最良の2番手予測。

同一ライン指数2位の問題:
  ライン内2位 = ライン戦術上はpivot1の直後につく「番手選手」
  しかし「番手選手」が3着以内に入るかどうかは
  ① pivot1が先行成功するか（外乱あり）
  ② ライン外の選手がより強い場合がある
  この不確実性をAIは学習済み → AI2位がライン外でも正当
```

## なぜ指数下位除外は効かないか

doc50（三連複買い目削減）と同型の問題:

```
指数最下位 = pred5的な役割 → ROI 111.4%★（市場の過小評価）
指数下位除外 = 「市場が正しく低評価した目」だと仮定するが
実際は「AIが低確率・市場が低オッズ→組み合わせ的に見逃されがち」な
高ROI目を削除している可能性がある。
```

R2（下位2名除外・VAL 51.8%）がR1（下位1名・VAL 62.3%）より更に悪化するのは、
除外するほど高ROI目が消えることを示している。

## 結論

| 戦略 | 判定 | 理由 |
|---|---|---|
| L1/L2（ライン軸2+除外） | **不採用** | 軸2変更で ROI 悪化・市場が既にライン関係を織り込み済み |
| R1/R2（AI軸2+除外） | **不採用** | 指数除外はdoc50のP2系と同型・VAL 51-62% |
| **S0 現行3点** | **維持** | 全変更バリアントで最良・変更に意義なし |
| **O10（doc50）** | **live検証候補** | 今回の「目削減」とは異なる軸（コンボオッズ10倍+足切り） |
