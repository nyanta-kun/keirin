## 3. W03 要約（交差条件の検証）

### 構造的非重複の発見

現行戦略（tier=SS/S/A）の採用条件（gap12 大 = 予測確率の1-2位差が大きい）と
upset Q4（top3_sum 小 = 予測確率が分散）は **鏡面関係**。

- 全≤6車の upset Q4 閾値（0.578）を使うと、戦略通過 304R 中に upset Q4 は **1R のみ（0.3%）**

## 5. クローズ理由（全不通過）

1. **波乱モデルが汎化しない**: VAL/HOLD AUC ≈ 0.57。≤6車（約5,600R）ではレース構造が均質で
   TRAIN の波乱パターンが OOS に汎化しない。標本数も不足（波乱レース約900R）。

2. **現行戦略は隠れた波乱回避フィルター**: tier 条件（gap12 大）と波乱型レース（top3_sum 小・gap12 小）が
   鏡面関係にあるため、「波乱×現行戦略」の交差は構造的に成立しない。
   波乱モデルを追加しても現行 tier 条件が既に同等の役割を担う。

3. **フォーメーション工夫に限界**: F2（BOX6）が HOLD 133% と若干高いが CI 下限 70%・6点投資・単発依存が疑われ
   採用根拠なし。「どのラインが崩れるか」の事前予測精度が市場オッズに折り込まれていると解釈される。

4. **doc02 の数字は完全無効**: 3バイアス + 波乱ラベル誤りの合成で、実証的価値がなかった。

5. **多重比較の観点**: upset 閾値変更・他の買式・他の特徴量組み合わせ等の変種は
   多重比較ノイズになるため追わない。

## 6. 構造的知見（市場効率・波乱予測の限界）

| 知見 | 内容 |
|------|------|
| 市場価格と的中確率の整合 | ≤6車でも三連複市場は適切に価格設定されており、公開情報から超過収益を得るのは困難 |
| 波乱ラベルの設計 | wt_odds は全組み合わせ格納のため MAX 払戻は使用不可。的中組み合わせ払戻（波乱率16%）が正しい定義 |
| tier 条件の副機能 | 現行 SS/S/A 選別は gap12 大を要件とし、確率分散型（波乱型）レースを自動排除する隠れたフィルター |
| upset × fav_mismatch の独立性 | φ≒0 で独立情報源だが、交差集合が小さすぎて実用的な検出力がない |
| 再開条件 | 波乱型レースへの参加は公開オッズを超える情報源（money-flow等）が必要。现時点では再開条件なし |
