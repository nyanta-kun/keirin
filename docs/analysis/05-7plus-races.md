# 7車以上レースの収益化検討 — 結論: 撤退維持（大標本で確定）

> 2026-06-08 / `scripts/exp_7plus_wt.py`・`exp_7plus_2car_wt.py` / 指数=eval model(OOS) / TRAIN 64,766R・TEST 6,559R（7+のみ・大半が7車、9車541R）/ 払戻=最終オッズ=上限値

## 背景
現行は ≤6車のみが賭け対象。7+（主に7車・9車）は組合せ数が多く従来「採算不可」とされてきたが、**特定条件で再現的に黒字化できるか**を大標本・train→OOS で再検証した。

## 結果: 全券種・全条件・全構造で回収率72〜80%（控除率の壁）

| 券種/構造 | 代表ROI(TEST) | 備考 |
|---|---|---|
| 三連複 std3 / wide5 / box4 / box5 | 77 / 75 / 72 / 69% | gap12↑でも不変(≥0.20で79%が最高) |
| 二車複 top2(1点) / box3 | 77 / 72% | |
| ワイド top2 / box3 | 74 / 72% | |

- **gap12≥0.20（強い本命）でも改善せず**（三連複std3 79%が天井）。`top3_sum`下位（波乱寄り）でも改善せず（〜81%）。
- 全て **train≈test で一致・CI狭**（例 ALL std3 test [74,81]）＝偶然でなく構造的。

## 解釈
- ≤6車を黒字化していたモデルのエッジ（特に Q1_loose で 1,000%超）は **7+ では消滅**。
- 7+は出走数増で組合せが多く市場が効率的、モデルのtop3ランキングだけでは**控除率(約25%→ROI上限〜75%)を超えられない**。
- = 「7+採算不可」は推測でなく**大標本で実証**された。≤6車への集中が正しい。

## 追検証: 三連単フォーメーション（`scripts/exp_7plus_trifecta_wt.py`）

「三連複(1位含む)が壁なら三連単で点数を絞り高配当を」という案も検証。指数1位の1着率(7+)=46%。

| 戦略(7+) | ALL TE | gap12<0.10 | gap12≥0.20 | 判定 |
|---|---|---|---|---|
| F1a 本命1着固定・相手234(6点) | 69% | 60% | 79% | × |
| F1b 本命1着固定・相手345穴(6点) | 73% | 73% | 67%(train86→test崩壊) | ×(非再現) |
| F2 本命を2-3着・波乱頭(8点) | 72% | 78% | 66% | × |

- **全フォーメーション・全条件で60〜86%・100%超は train/test 両立せず**。
- 「本命を2-3着に置く波乱頭(F2)」で favorite-longshot bias を突く狙いも**控除率を超えず**（66-78%）。

## 追検証: 7+専用モデル（`scripts/exp_7plus_model_wt.py`・2026-06-09）

「プールモデルでなく7+専用モデルなら？」を検証（7+のみで学習・train2023-07〜2025-11/val2025-12〜2026-02/test2026-03〜）。

| | 7+専用モデル | プールモデル(全車) |
|---|---|---|
| AUC(test/7+) | **0.7777** | **0.7880**（プールが上）|
| ROI(全条件×全構造) | 67〜76% | 75〜80% |

- **専用モデルのAUCはプール未満**（データ少で≤6車からの転移汎化を失い劣化）。ROIは gap12≥0.20・top3_sum下位等の全条件×std3/wide5/box4/box5 で **67〜76%・val≈test・100%超皆無**。
- **含意**: 7+で100%を超えられないのは「モデルの問題」でなく「市場構造」。プールモデルは7+を AUC0.788 と十分上手くランク付けできており、それでも控除率(〜75%)を超えないのは大フィールドの市場効率＋組合せ過多のため。**より良いモデルでは解けない**。

## 追検証2: 極端な選別×ナロー買い（`scripts/exp_7plus_selective_wt.py`・2026-06-09）

「的中率でなく、購入レースと買い目を絞ればROI100%超では？」を極限まで検証（pooled model・gap12/top3_sumを極端閾値・買い目を1〜3点に）。

| 選別 | std3 TRAIN | std3 TEST(n) | 判定 |
|---|---|---|---|
| gap12≥0.30 | 84% | 98%(378) | TR<100% |
| gap12≥0.40 | 84% | 95%(45・CI[55,148]) | TR<100%・小標本 |
| top3_sum≤p10(開10%) | 82% | 107%(338) | TR<100% |
| top3_sum≤p05(開5%) | **92%** | 123%(148・CI[63,203]) | TR<100%（天井92%）|

- **大標本TRAINはどの極端スライスでも100%未満**（最も開いた5%でも92%が天井）。TESTの散発的100%超(t1 156%等)はn≤148・CI極広・TRAIN矛盾＝**ノイズ**。
- ナロー買い(1点/三連単1点)も改善せず（三連単1点は42-72%と悪化）。
- = 7+は選別・絞り込みを極限化しても控除率を超えない。≤6車で同手法が効いた(Q1_loose 1100%)のと対照的＝**7+は市場が効率的で妙味ポケットが存在しない**。

## 結論
**7車以上は三連複・三連単・専用モデル・極端選別×ナロー買いの全てで控除率の壁(天井〜92%)を超えられない＝撤退維持で確定**。賭け対象は **≤6車**。7+再検討の価値は「公開情報を超える情報/未発見の構造的非効率」を得た場合のみ。

（注: 検証は最終オッズ上限値。実運用ではさらに下振れ＝7+の不利はより強い。）
