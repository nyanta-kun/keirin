# 7車以上レースの収益化検討 — 結論: 撤退維持（大標本で確定）

> 2026-06-08 / `scripts/exp_7plus_wt.py`・`exp_7plus_2car_wt.py` / 指数=eval model(OOS) / TRAIN 64,766R・TEST 6,559R（7+のみ・大半が7車、9車541R）/ 払戻=最終オッズ=上限値

## 背景
現行は ≤6車のみが賭け対象。7+（主に7車・9車）は組合せ数が多く従来「採算不可」とされてきたが、**特定条件で再現的に黒字化できるか**を大標本・train→OOS で再検証した。

## 結果: 全券種・全条件・全構造で回収率72〜80%（控除率の壁）

| 券種/構造 | 代表ROI(TEST) | 備考 |
|---|---|---|
| 三連複 std3 / wide5 / box4 / box5 | 77 / 75 / 72 / 69% | gap12↑でも不変(≥0.20で79%が最高) |
| 二車複 top2(1点) / box3 | 77 / 72% | |
| ワイド top2 / box3 | 74 / 72% | |

- **gap12≥0.20（強い本命）でも改善せず**（三連複std3 79%が天井）。`top3_sum`下位（波乱寄り）でも改善せず（〜81%）。
- 全て **train≈test で一致・CI狭**（例 ALL std3 test [74,81]）＝偶然でなく構造的。

## 解釈
- ≤6車を黒字化していたモデルのエッジ（特に Q1_loose で 1,000%超）は **7+ では消滅**。
- 7+は出走数増で組合せが多く市場が効率的、モデルのtop3ランキングだけでは**控除率(約25%→ROI上限〜75%)を超えられない**。
- = 「7+採算不可」は推測でなく**大標本で実証**された。≤6車への集中が正しい。

## 追検証: 7+専用モデル（`scripts/exp_7plus_model_wt.py`・2026-06-09）

| | 7+専用モデル | プールモデル(全車) |
|---|---|---|
| AUC(test/7+) | **0.7777** | **0.7880**（プールが上）|
| ROI(全条件×全構造) | 67〜76% | 75〜80% |

## 量(1日の対象数)の観点 — 7+は不要

7+を望む動機は「1日の対象を増やしたい(1-2件/日)」だが、≤6車だけで既に充足:
- 2026-03〜06(99日)の≤6車・gap12≥0.06: **推奨(≥5倍)1.90R/日**・B(3-5倍)1.05R/日・見送り1.78R/日。
- 推奨0件の日は12/99(12%)のみ＝88%は1件以上。推奨+B 0件は5/99(5%)。
- 最近の0件続き(6/8・6/9)は12%の静かな日の偶然連続。
- **量を増やすなら +EV の Bランク(ROI~116%)を買い目に含める**(推奨+B=~3件/日・0件日5%)のが正解。**7+(-EV ~92%)で量を稼ぐのは「お金を払って参加を増やす」ことになり非推奨**。

## 追検証5: ライン構造ベースの波乱軸（「1位を1着で買わない」）（`scripts/exp_7plus_lineaxis_wt.py`・2026-06-09）

## 追検証8: 本命(pred1位)バスト狙い＝1位を外し残りで組む（軸落ち46%の高配当側）（`exp_7plus_favbust_wt.py`・`exp_7plus_favbust2_wt.py`・2026-06-09）

「軸落ち46%のうち1位飛びは人気が飛ぶため高配当。1位を買い目から外し相手も残りから選定」。
pred1位は7+で~80%が3着内＝1位除外買いはその80%が自動外れ(コスト)。勝負は①飛ぶ20%を予測できるか②飛んだ時に残りを当てるか。

- **①バストは部分的に予測可能**: ALL 20% → r1_prob≤p25で**34.7%**・≤p5で更に集中（モデル自身の1位への低信頼が最良の予測子）。gap12低/n_senko多も微増。1位が逃(style=0)は逆に低バスト17%。
- **②1位除外買いROI（最良=r1_prob低選別×高オッズ帯）**: バスト選別を強めるほどROIは 49-66%(ALL)→ **TRAIN天井90%** へ上昇するが**100%を越えない**。

| 買い目×帯×選別 | TR_ROI | TE_ROI(n) | TR_R |
|---|---|---|---|
| box r2-6 × [40+] × r1p≤p5 | **90%** | 50%(172) | 3229 |

- **解釈**: 仮説の前提(飛び＝高配当・飛びは予測可)は正しいが「だから勝てる」は不成立。**1位を外す＝高オッズ組合せを買う＝favorite-longshot bias(穴は過剰人気＝割高)の罰**を受ける。モデルのバスト予測が49%→90%まで取り返すが、控除率＋穴の割高分を越えられずTRAIN天井90%。**7+で最も健闘した戦略だが、それでも壁の手前**。
- (副産物: r1_probはバストの良い予測子＝モデルの自己信頼度として機能。≤6車の信頼ゲートに転用余地あり。)

## 追検証9: 本命バストの「個別精度」分解（払戻と突き合わせる前段）（`exp_7plus_bust_accuracy_wt.py`・2026-06-09）

ROIでなく的中精度を、検出と軸選定に分けて確認。

**A. バスト検出精度（1位が3着外を当てる）**: 全体バスト20%。
- baseline(1-r1_prob) TEST AUC **0.713** / 専用LGBM分類器(10特徴) TEST **0.716**（TRAIN0.778=過学習）。
- **専用化の余地ほぼ無し**＝バスト検出はモデル自己信頼度r1_probで出し尽くし（重要度もr1_probが圧倒）。キャリブレーション良好(予測十分位で実バスト1%→40%単調)。**飛びやすいレースのランク付けは可能だがAUC0.71＝既約ノイズ大**（波乱の本質）。

**B. 波乱時の軸選定精度（1位バスト時に車券内に来るのは誰か）= 重要な発見**:

B1 pred順位別top3率(バスト時): 2位69.6%/3位62.8%/4位56.4%/5位46.2%（pred順序は保たれるが各順位とも非バスト時より上昇）。

B3 **1位ライン関係別top3率(バスト時)**: **同ライン(番手等)34.9% / 別ライン52.9%**。

B4 **相手第1候補別top3率(バスト時・1レース1候補)**:
| 候補 | top3率 |
|---|---|
| **別ライン最上位** | **76.1%** ★最良 |
| pred2位(r2・現行軸) | 69.6% |
| 両or追 最上位 | 66.0% |
| 追(差し)最上位 | 60.5% |
| 1位ライン番手(同ライン最上位) | 43.0% 最悪 |

- **核心**: 1位(多くは逃)が飛ぶ時はその番手(同ライン)も道連れで崩れる(同ライン34.9%)。車券内に残るのはまくった別ライン(52.9%)。**現行「pred2位を第2軸」は2位がしばしば1位の番手＝崩れ道連れで最適でない**。**別ライン最上位を軸にすると 69.6%→76.1%(+6.5pp) 的中精度向上**。脚質より「ライン」が効く(B2: 逃52.8/両49.9/追44.8とライン差ほど大きくない)。
- = ユーザー仮説「1位が飛ぶ時の相手は2位とは限らない」は**正しい**。バスト時の軸は「別ライン最上位」が正解。
- **未実施(次段)**: この精度改善(別ライン軸)を反映した1位除外買いのROI再検証。追検証8のbust-fade(TRAIN天井90%)は**pred順/box=ナイーブな軸**で、別ライン構造を使っていない。別ライン軸でROIが90%の壁を越えるかは未検証＝7+の最終判定は保留。
