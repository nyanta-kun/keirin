## 解釈・結論

### gami フィルタが 7+車でも機能する理由

| フィルタなし | gami≥5倍あり |
|------------|------------|
| 市場人気の高い「本命明確」レース多数 | 市場が不確実とみるレース = 高配当帯 |
| ROI 86-91%（市場が正しい） | ROI 116-130%（モデルのエッジが生きる） |
| 全体の 73%（14666R/VAL） | 全体の 27%（4664R/VAL） |

→ **gami≥5倍 は ≤6車 同様に 7+車でも「市場不確実性フィルタ」として機能**  
→ 前回クローズは gami フィルタなしの全体評価 = 条件付き分析が欠けていた

### n_lines=7（単騎集合）の特殊性

- 全選手が単独出走（チーム戦略なし）→ 個人能力の純粋勝負
- 得点上位者（最高得点者）が top3 に入る確率 88%（事前確認）
- モデルが得点rank1に一致 + gap12≥0.07 → VAL 690%★（ただし月別波大・n小）

### n_lines=4 が苦手な理由

- 4ライン混戦 = 最も複雑な展開パターン
- VAL ROI 95.8%（gap12≥0.07 でも不通過）
- 市場も予測困難 = 高配当だが的中確率も低下
