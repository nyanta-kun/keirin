## 結論

### 5R/日目標は現行データでは達成不可能

| 理由 | 詳細 |
|------|------|
| **gami フィルタが必須** | gami < 5倍（B-rank・no-gami）の trio ROI は両期間とも 77〜87%。gami フィルタなしでは収益が出ない |
| **ワイド W12 は不採算** | C0 対象レースでも W12 ROI は 85〜88%（VAL・HOLD 両方）。市場効率が高い |
| **プール上限が 4R/日** | ≤6車 レース自体が ~4R/日しかなく、5R/日の物理的限界が存在する |
| **7+車はクローズ済み** | 7台以上は 2026-06-09 にクローズ確定（全 19 軸・EV 原理・独立再現で市場効率確定） |

### なぜ gami フィルタが効くのか

- gami（最低三連複オッズ）は「市場の不確実性」の代理変数
- gami≥5倍 = 市場が3択全組み合わせを5倍以上に評価 → モデルのエッジが生きる空間
- gami < 5倍 = 重厚な本命オッズ → 市場がすでに正解に近い評価 → モデル優位なし

### なぜワイドは不採算か

- W12 は「1・2着予想の両者がトップ3に入る」 = 市場が最も分析しやすい賭け式
- モデルが確信する上位2着（gap12≥0.07）は市場でも人気馬 → W12 オッズは低くなりやすい
- W12 odds≥2.5 対象レースは全体の 5% のみ（99%は 2.5 倍未満で対象外）
- 残る 5% のレースも ROI < 100% → 市場の評価が正しい

## 推奨事項

1. **C0 を維持する（0.25R/日）**: 現時点で唯一の採算戦略。変更不要。
2. **5R/日目標は再設定が必要**: 現行データ・ルールの枠内では達成不可能。
   - ≤6車プールの上限が ~4R/日（5R/日には届かない）
   - ROI正の戦略はその 6% しか使えない（0.25R/日）
3. **gap12≥0.05 の trio（T2a）**: VAL 206.6% だが HOLD n=3 で判断不能。live 観察を継続（約 6〜12 ヶ月で 12R 以上蓄積後に再評価）
4. **新情報源が唯一の拡張路**: money-flow（doc23・約 9 ヶ月待ち）・試走タイム（難易度高）など、市場未価格化の情報のみが新たな採算レースを生む可能性がある
