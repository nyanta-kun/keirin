# 7+車 推奨ランク ルール・評価結果

**対象**: 7車以上レース  
**買い目形式**: 三連複  
**検証モデル**: lgbm_wt_train_only（TRAIN 2023-07-01〜2025-06-30、リーク無し）  
**バックテスト期間**:
- VAL: 2025-07-01〜2026-02-28
- HOLD: 2026-03-01〜2026-06-12

---

## ランク一覧

| ランク | 条件 | 買い目 | HOLD ROI | VAL ROI |
|--------|------|--------|-----------|---------|
| **SSランク** | gap12≥0.07 かつ ガミ目カット後 残り≤3点 | 残存目のみ（1〜3点） | **137.0%★** | **159.8%★** |
| **Sランク** | 全目gami≥5倍 + gap12≥0.10 | 全相手流し（全点） | **~143%★** | — |
| **Aランク** | 全目gami≥5倍 + gap12∈[0.07, 0.10) | 全相手流し（全点） | **~138%★** | — |

> ★ = HOLD期間リーク無しバックテスト合格基準（>100%）  
> S/A合算: VAL 129.9%★ (3,143R) / HOLD 138.3%★ (1,381R) / 12.9R/日 [doc48]

## 用語定義

| 用語 | 定義 |
|------|------|
| **gap12** | AI予測確率の1位-2位間の差。大きいほど軸の優位性が高い |
| **gami（ガミ）** | 全買い目のうち最低オッズ。5倍未満の目はほぼ損確定 |
| **ガミ目カット** | 各三連複目のオッズが5倍未満のものを除外する操作 |
| **残り≤3目** | カット後に残った有効な買い目が1〜3点であること |
| **HOLD ROI** | 検証期間（2026-03〜06）の回収率。モデル学習期間外で算出 |

## SSランク 詳細

### 条件

```
7車以上 AND gap12 ≥ 0.07
→ 各買い目（軸1-軸2-相手）のオッズ < 5.0倍 を除外
→ 残り目数が 1〜3点 の場合のみ購入
```

## S/Aランク 詳細

### 条件

```
7車以上
AND 全買い目オッズ min ≥ 5.0倍（gami≥5倍）
AND gap12 ≥ 0.07
→ gap12 ≥ 0.10 → Sランク
→ gap12 ∈ [0.07, 0.10) → Aランク
```

## 運用仕様

| 項目 | 仕様 |
|------|------|
| 生成時刻（日中） | 毎朝 8:00（cron、daily_picks_wt.sh） |
| 生成時刻（夜の部） | 毎日 16:00（cron、evening_picks_wt.sh）→ 19時以降レース |
| 1点あたり投資額 | 100円 |
| SSランク投資額 | 100〜300円/レース（残り目数×100円） |
| S/Aランク投資額 | 全相手点数×100円/レース |
| Discord通知 | 朝の部・夜の部 それぞれ通知 |
| 翌朝採点 | notify_results_wt.py が確定結果を採点・通知 |

---

## 注意事項

- バックテスト結果は**過去データ**によるもの。将来の回収率を保証しない
- 3バイアス修正済み（欠車生存バイアス・≤6車混入・週次再学習リーク）
- live検証開始: 2026-06-16〜（目安100R=約1週間で初回判断）
- オッズは生成時点のもの。購入前に最新オッズを確認推奨
- 欠車（出走取消）があった場合、軸欠車はレース無効（返還）、相手欠車はその目のみ除外
