# タスク① ks→wt 追加特徴量の効果検証 と 閾値見直しの要否

**最終更新**: 2026-06-08 / 担当: 分析（report-only。本番モデル・戦略コードは未変更）

## 要約（結論先出し）

- **AUC は ks→wt 追加特徴で着実に上がる（コアのみ 0.7523 → フル39 0.7771）が、ROI とは連動しない**。テスト期間（810R対象）の層別合計ROIは「コア+ライン群のみ」が最良（431%）で、フル39（349%）はそれより低い。**AUC↑≠ROI↑** がここでも再現。
- **ライン群②が最大の純増要因**（合計ROI 350%→431%, +81pp / AUC +0.0168）。セクター③・AI印④は単独では合計ROIをむしろ下げる（過剰適合・本命寄せ）。rolling⑤はAUCを押し上げるがROI寄与は限定的。
- **AI印（prediction_mark）は feature importance では gain の42%を占める最大特徴**だが、ablationで外すと AUC は微減（0.7771→0.7714）の一方 **合計ROIは 349%→386% と上昇**。市場人気の代理＝本命に寄せる＝低配当化しており、収益面ではマイナス寄与。
- **波乱検出**: 「実1-2着が別ライン＝崩れ」レースのROIが「通り」を大きく上回る構造を再現（A層 通り124%/崩れ308%、フル39）。ライン特徴投入で崩れS層ROIが241%→443%に改善し、波乱の取りこぼしが減っている。
- **閾値**: train で最良だった候補の多くは OOS(test 810R)で現行ks閾値（合計349%）を明確には超えない。**現状維持を推奨**。唯一 `gap_lo=0.08` 系が OOS でも 397〜414% と上振れするが、対象R数が446→359に減り、過学習リスク・サンプル減を踏まえると「採用は要追試」止まり。

> 注意: 本レポートのROIは全て **最終データでのbacktest値＝実運用ROIの上限値**。実運用では大幅に下振れする（CONTINUATION記載の実測=49%/1週間）。断定不可。

## 1. 特徴重要度（本番 lgbm_wt, gain）

39特徴を5群に集計（gain合計に対する%）:

| 群 | 特徴数 | gain合計% |
|----|------|----------|
| ④AI印 (prediction_mark) | 1 | **41.7%** |
| ①コア | 18 | 40.2% |
| ②ライン | 6 | 9.9% |
| ⑤rolling | 9 | 7.2% |
| ③セクター | 5 | 1.0% |

上位特徴（gain%）: `prediction_mark` 41.7 / `score_z` 17.2 / `score_rank` 14.4 / `line_frac` 5.8 / `line_pos` 2.0 / `race_point` 2.0 / `quin_6m` 1.7 / `top3_6m` 1.6 / `period_norm` 1.5。

- AI印 1特徴だけで gain の4割超 → モデルは winticket の AI印（市場人気代理）に強く依存。
- ②ライン群は `line_frac`（gain 5.8%）が突出。ライン規模比率がレース内序列の有力シグナル。
- ③セクター群（s/h/b_count, ex_*）は gain わずか1.0%＝ほぼ寄与なし。

## 2. Ablation（最重要）

**設定**: 同一データ・同一分割。train `2023-07-01`〜`2025-09-01`（59,437R）→ test `2026-03-01`〜（7,342R、うち層別対象は下表R数）。`train_lgbm(feature_cols=…, weight_col="w_field")`、`run_tiered_backtest_wt` 相当の SS/S/A 3点買い（本番 `_assign_tier` 完全再現: SS gap12≥0.15&ratio<1.3 / S gap12≥0.15&ratio[1.3,1.6) / A gap12[0.06,0.15)、6車以下）。ROI括弧内は層の対象R数。

| 構成 | Test AUC | SS_ROI(R) | S_ROI(R) | A_ROI(R) | 合計ROI(R) |
|------|---------|-----------|----------|----------|------------|
| (a) コアのみ（18特徴） | 0.7523 | 1132%(81) | 176%(129) | 225%(266) | **350%(476)** |
| (b) +ライン②（24） | 0.7691 | 1926%(60) | 320%(107) | 207%(260) | **431%(427)** ←最良 |
| (c) +ライン+セクター②③（29） | 0.7699 | 1489%(55) | 300%(104) | 244%(264) | 392%(423) |
| (d) +ライン+セクター+rolling②③⑤（38） | 0.7714 | 1780%(58) | 336%(101) | 158%(267) | 386%(426) |
| (e) フル39（=本番相当） | 0.7771 | 1113%(56) | 387%(108) | 212%(282) | **349%(446)** |
| (f) フル−AI印（38） | 0.7714 | 1780%(58) | 336%(101) | 158%(267) | 386%(426) |

### 純増分の読み取り

- **ライン群②**: 最大の収益純増。合計ROI +81pp（350%→431%）、AUC +0.0168。特に SS（1132%→1926%）・S（176%→320%）を大きく改善。**投入価値が最も明確な追加群**。
- **セクター群③**: (b)→(c) で合計ROI **431%→392% と低下**（AUCは+0.0008と誤差）。gain寄与1%とも整合し、**ノイズ/軽い過適合**。収益貢献なし。
- **rolling群⑤**: (c)→(d) で AUC +0.0015 だが合計ROI 392%→386%（ほぼ横ばい）。A層ROIは244%→158%へ低下。AUC寄与はあるがROIには結びつかず（既知の「rolling +12〜14pp」はより古い分割での結果で、本分割では中立〜微減）。
- **AI印④**: (d/f)→(e) で AUC +0.0057（最大のAUC寄与の一つ）だが**合計ROI 386%→349% と低下**。importanceトップなのにROIを下げる典型的な「予測精度↑＝本命寄せ↑＝低配当化」。

### AUC↑ がROI↑に繋がっているか → **No（明確に非連動）**

AUC順位（a<b<c<d=f<e）と合計ROI順位（b>c>d=f>a≒e）は一致しない。**AUC最良のフル39が合計ROI最下位**。本プロジェクトの「AUC↑≠ROI↑」が今回も成立。収益の鍵は「全体の判別精度」でなく「波乱配当を取れる上位序列の付け方」。

> 留意: 各層R数は数十〜数百Rと小さく、SS層（55〜81R）のROIは万車券1本で大きく振れる。層別ROIの差は点推定であり信頼区間は広い。

## 3. 波乱検出精度（ライン通り vs 崩れ）

**ラベル**: 各レースの実1着・実2着が**同一 line_group なら「通り」、異なれば「崩れ（波乱）」**。test期間・本番 `_assign_tier` で層を割り当てた上で集計。

### フル39モデル

| 層 | 通り R/的中/ROI | 崩れ R/的中/ROI |
|----|----------------|----------------|
| SS | 29R / 7 / 314% | 27R / 6 / **1936%** |
| S | 35R / 22 / 270% | 73R / 39 / **443%** |
| A | 146R / 89 / 124% | 136R / 62 / **308%** |

→ 全層で **崩れ＞通り**。CONTINUATION記載の「A層 通り158%/崩れ278%」（旧モデル）と同方向で再現（数値は分割・モデル差で多少ずれ、今回は通り124%/崩れ308%）。**収益源は波乱（崩れ）側**という結論を再確認。

### コアのみモデル（ライン特徴なし）との比較

| 層 | コアのみ 通り/崩れ ROI | フル39 通り/崩れ ROI |
|----|----------------------|---------------------|
| SS | 682% / 1442% | 314% / 1936% |
| S | 107% / 241% | 270% / 443% |
| A | 168% / 279% | 124% / 308% |

- **ライン特徴は崩れS・崩れSSのROIを押し上げる**（S崩れ 241%→443%、SS崩れ 1442%→1936%）。波乱レースで上位2車をより的確に拾えている。
- 一方コアのみは A層・崩れ側がフル39と同等以上（279% vs 308%は僅差、的中数はコア71/138 > フル62/136）。**ライン特徴がなくても rolling/得点で波乱の的中自体は取れている**（ks が詳細ライン無しで勝てる傍証と整合）。
- 総じて **ライン特徴②は「波乱レースを正しく層別・pivot配置する」方向に効いている**が、A層では決定的でない。

> 留意: 崩れ側は配当が高く分散が大きい。崩れSS 27R/的中6 など小標本で、ROI 1936% は1〜2本の万車券依存の可能性が高い。

## 4. 閾値再最適化（train グリッド → OOS test 検証）

**手順**: フル39モデルの pred_prob で、train(6,849R対象)で gap_lo（A下限）/ gap_hi（SS・S vs A 境界）/ ratio_ss（SS vs S 境界）/ ratio_s（S 足切り）をグリッドスイープ → 合計ROI上位候補を test(810R対象) で検証。現行ks閾値 = (0.06 / 0.15 / 1.3 / 1.6)。

### train 期間 合計ROI 上位候補

| gap_lo | gap_hi | r_ss | r_s | train合計ROI | train_R |
|--------|--------|------|-----|-------------|---------|
| 0.08 | 0.12 | 1.1 | 1.6 | 545% | 2909 |
| 0.08 | 0.12 | 1.3 | 1.6 | 538% | 2909 |
| 0.08 | 0.12 | 1.5 | 1.6 | 529% | 2909 |
| 0.08 | 0.15 | 1.1 | 1.6 | 501% | 3178 |
| 0.06 | 0.12 | 1.1 | 1.6 | 497% | 3475 |
| **現行 0.06/0.15/1.3/1.6** | | | | **459%** | 3744 |

### OOS test 検証（現行 vs 候補）

| 設定 | train合計ROI | TEST合計ROI(R) | TEST SS / S / A |
|------|-------------|----------------|------------------|
| **現行 0.06/0.15/1.3/1.6** | 459% | **349%(446)** | 1113%(56) / 387%(108) / 212%(282) |
| 0.08/0.12/1.1/1.6 | 545% | 397%(359) | 1470%(32) / 406%(182) / 212%(145) |
| 0.08/0.12/1.3/1.6 | 538% | 367%(359) | 998%(62) / 304%(152) / 212%(145) |
| 0.08/0.12/1.5/1.6 | 529% | 414%(359) | 722%(150) / 196%(64) / 212%(145) |
| 0.08/0.15/1.1/1.6 | 501% | 374%(393) | 1666%(29) / 505%(135) / 182%(229) |
| 0.06/0.12/1.1/1.6 | 497% | 390%(412) | 1470%(32) / 406%(182) / 247%(198) |
| 0.08/0.15/1.3/1.6 | 495% | 350%(393) | 1113%(56) / 387%(108) / 182%(229) |
| 0.06/0.12/1.3/1.6 | 492% | 364%(412) | 998%(62) / 304%(152) / 247%(198) |

### 読み取り

- 現行ks閾値は train 459% / OOS 349%（446R）。多くの候補は OOS で +15〜65pp 程度の上振れを見せる。
- **共通パターン**: gap_lo を 0.06→0.08 に上げる（買い目を絞る）と train/OOS とも ROI 改善（が R数は減る）。gap_hi を 0.15→0.12 にすると SS/S 振り分けが変わり ROI が上がる傾向。
- ただし OOS で最良の `0.08/0.12/1.5/1.6`（414%）は SS判定が ratio<1.5 に緩み SS が150R/722% へ（≒S層化）。これは**層定義の意味が変質**しており、単純な閾値最適化というより層再設計。
- **過学習リスク**: train上位 = OOS上位 の対応は弱い（train1位 545% は OOS 397%で4番手相当、train3位 529% が OOS 1位 414%）。train で選んだ最良がOOSで再現する保証は薄い。**サンプル(test 810R全体、層別では数十〜数百R)も小さく、点推定の差は誤差圏内**。

## 5. 推奨アクション（提案・report-only）

1. **セクター群③は削除候補**。gain 1%・ablationでROIを下げる。次回再学習時に外して再検証する価値あり（38→33特徴）。**FEATURE_COLS_WT 変更＝要再学習**（CLAUDE.md ルール）。
2. **AI印④の扱いを再検討**。importanceトップだがROIを下げる。完全除去より、AI印が強い＝本命濃厚レースを「低配当としてA層から外す/賭けない」フィルターに転用する方が筋が良い可能性。実験提案にとどめる。
3. **ライン群②は維持。波乱（崩れ）側の収益貢献が明確**。波乱予測強化の方向（line_group の崩れ確率を別途モデル化し層に反映）が次の伸びしろ。
4. **閾値**: gap_lo を 0.06→0.08 に上げる案は train/OOS とも一貫してプラスで、最も再現性が高い候補。ただし**採用は追加のOOS期間（複数ホールドアウト）で再検証してから**。現時点では「有望だが未採用候補」。gap_hi/ratio の変更は層定義変質を伴うため非推奨。

## 6. 注意点（上限値・過学習リスク）

- 全ROIは最終データの **backtest値＝実運用ROIの上限**。実運用は大幅下振れ（実測49%）。
- 層別R数が小さく（SS 27〜150R）、ROIは万車券依存で分散大。差は点推定で信頼区間は広い。
- train で良い閾値が OOS で再現しない事例が今回も発生。**「OOSで再現しない改善は不採用」原則**を堅持。
- 本検証は単一のtrain/val/test分割。結論の堅牢性確認には複数分割・ローリングホールドアウトが望ましい。
