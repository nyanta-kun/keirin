## 検証の動機

ユーザー提案：「三連複推奨レースの一部について、軸の着固定・不要買い目削除によるフォーメーション三連単に転換し、特別推奨レースとして提示できないか」

## 解釈

### なぜ T7 が唯一 Phase2 ボーダーを通過するか

1. **gap12≥0.10**: pred1 がある程度絞り込まれ 1着率が上昇
2. **gap23≥0.05**: pred2 が pred3 に対して優位 → 2着固定の精度が上がる（P1&P2_exact=20.0%）
3. **2点（pred3・pred4 のみ）**: pred5（5番手評価）を除去して資金効率改善
4. **三連単オッズ**: 三連複より大幅に高オッズ → 的中率 10-17% でも ROI が高水準

### 三連複との比較

- S0 三連複: 0.10-0.12帯 hit率 41.7% / ROI 73.8%（的中多いが還元率低い）
- T4/T7 三連単: hit率 7-17% / ROI 114-229%（的中は少ないが着順固定が当たると高還元）

三連単は「低的中率×高払戻」の構造。勝てる条件が限定的（gap23 追加フィルタが鍵）。

### 三連単が全体的に劣る理由（doc33 の結論を本実験が裏付け）

- 市場は三連単のオッズも効率的に設定 → 単純な 1着固定はエッジを持てない
- 条件絞り込み（gap12+gap23）によって初めてエッジが生まれる可能性がある

## 実運用設計（案）

T7 を「特別推奨」として live 追跡する場合:

```
条件: ≤6車 AND ガミ≥5倍 AND gap12≥0.10 AND gap23≥0.05
買い目: pred1→pred2→pred3 (1点) + pred1→pred2→pred4 (1点) = 計2点
想定頻度: 約0.3R/日（10R/月）
初回判断: 50-60R 蓄積後（約5〜6ヶ月）
```

**現行 三連複推奨との関係**: 同レースで三連複も三連単も提示するか、三連単のみにするかはユーザーの判断。三連複(3点)+三連単(2点)=5点の複合購入も可能。
