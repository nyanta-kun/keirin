# 波乱の解剖 — 条件・指数×ライン（探索分析）

> 2026-06-08 / `scripts/analyze_upset_anatomy_wt.py` / 指数=eval model(OOS寄り) / 2025-06〜2026-06・≤6車・gap12≥0.06・2,171R / 払戻=最終オッズ=上限値

「軸1位ありき」を仮定せず、波乱レースの正体を解剖した。目的: 波乱の発生条件と、発生時の指数×ラインの関係を把握し、買い目戦略の土台にする。

---

## 0. 要約

1. **波乱(1-2着が別ライン=line_break)は全体の51%**で発生。`top3_sum` 四分位がよく層別する（Q1_loose 68% → Q4_chalk 39%）。波乱帯ほど三連複配当が高い（Q1中央4,230円 vs Q4 420円）。
2. **波乱発生の最大の事前条件は `n_lines`（ライン数）**: 2本33% → 3本44% → 4本62% → 5本74% → 6本以上100%。`単騎数(n_iso)` も同方向。`gap12≥0.25` の超本命偏重レースも逆に割れやすい(63%)。
3. **波乱でも指数1位は83〜85%が3着内**＝アンカーとして堅い。一方、波乱時は **実top3の約1.8人が本命ライン外**（非波乱は0.9人）。
4. **波乱の"伏兵"（実top3で最も指数が低い1名）は、ほぼ非本命ライン(88%)の「ライン先頭」(66%)で、指数rankは3〜4位が58%**（深い大穴rank5-6は少ない）。＝指数はまだ効いており、狙うべきは「非本命ラインの先頭・指数3-4位」。

→ 含意: 波乱レース（n_lines多・top3_sum低）では、**指数上位の鉄板2軸ではなく「非本命ラインの先頭格(指数3-4位)」を絡めた買い目**が、的中とライン崩れ高配当の両取りに繋がる可能性。

## 4. 戦略への示唆（次段）

- 狙うレース: **n_lines が多い / top3_sum 低い（Q1_loose）** レース。
- 買い目の発想: 指数1位（83%）を信頼アンカーにしつつ、**「非本命ラインの先頭・指数3-4位」を相手に厚く**。鉄板2軸(指数1-2位)から外し、ライン崩れの高配当を取りにいく。
- 検証必須: 上記を実オッズでOOSバックテストし、的中率低下を配当が上回るか（ネット+EVか）を確認。機械的逆張りは過去に反予測的だった点に留意（今回は波乱予測×ライン×指数の条件付き）。

## 5. 付随して判明した重大バグ（修正済）

- **脚質特徴 `style_enc` が全件 -1（死んでいた）**。winticketの `style` 値は実際には `逃/両/追`（各18.5万/17.1万/32.6万件）だが、`feature_wt._STYLE_MAP` が旧表記「先行/捲り/差し/追い込み」を前提でキー不一致→全件 -1。
- 修正: `_STYLE_MAP` に `逃=0(先行)/両=1/追=2(追込)` を追加し再学習。
- **精度影響は中立**（holdout AUC 0.7777→0.7778）。＝ 脚質情報は既に s_count/h_count/b_count 等で実質取り込まれており、生の3値脚質の限界寄与は小さい。とはいえ死んだ特徴の正常化は正しく、波乱解剖の脚質次元（上記§3）が分析可能になった。配信モデルにも反映（full-refit再学習）。
