> ユーザーの問い: ①「指数の分布の形（1車抜け/2車抜け/階段/拮抗）×脚質×ラインは考慮されているか・効くか」

## 結論（要約）
1. **形×脚質×ラインの大部分は「モデルが既に知っている」**（較正残差≈0）＝特徴追加不要。構造は実在する（同ライン2車は残存75% vs 別ライン62%等）が、**実在する構造は市場もモデルも織込み済**でエッジにならない。
2. **例外＝「指数下位の単騎逃げ」**: 拮抗レースで唯一の逃げ車が指数4位以下のとき、その車のtop3をモデルが系統的に過小評価（**残差 TRAIN+7.1%/n=244・TEST+27.6%/n=20**）。→ `is_lone_nige` 特徴の追加検証へ（`exp_lone_nige_feature_wt.py`）。
3. **★★最大の発見＝「本命人物の不一致」**（モデル1位≠市場本命）: 現行戦略ROIが**TRAIN 1168%**[884,1491]最大除1101%(n=1108) / **TEST 576%**[365,809]最大除521%(n=116)。一致時(210/336%)の3〜5倍。top3_sumと相関-0.185＝**既知レバーと独立**。→ `fav_mismatch` タグ実装・live前向き検証へ。

## 実験1: 分布形状×脚質×ライン（事前登録H1-H7＋規律チェック）

形の分類: pred_prob降順のギャップ構造 G=[p1-p2,p2-p3,p3-p4]、spread=ΣG。
拮抗(spread<q30=0.331)30% / 2車抜け21% / 3車抜け20% / 1車抜け18% / 階段11%。

| 仮説 | 結果 | 判定 |
|---|---|---|
| H1 1車抜け×脚質×逃げ被り | 実績差は実在（追79% vs 逃88-93%）だが予測も同差＝残差+0〜5%不安定 | **モデル既知** |
| H2 2車抜け×ライン | 同ライン残存75.1%>別ライン62.1%(TR)＝構造実在。だが市場織込み済（中央320円vs730円）でROIは別ライン側が上。「別ライン逃逃共倒れ」はTR1365%→TE87%で崩壊 | 構造実在・エッジ無し |
| H3 拮抗×逃げ人数 | 逃げ1人が最良(TR256/TE353%)だがCI重複。**P4-3が例外（下記）** | 弱い |
| H4 抜け追込×孤立 | **該当0件**（≤6車で抜けた追込は必ずライン保有） | 空集合 |

- **P4-3（指数下位の単騎逃げ）= 唯一の較正破れ**: その車top3実績 TRAIN44.7% vs 予測37.6%（+7.1%・n244）/ TEST65.0% vs 37.4%（+27.6%・n20）。同方向・「楽に主導権を取る格下単騎逃げ」をモデルが過小評価。n_senko(人数)はあるが「誰が」の交互作用不足。
- 第2陣: 2車抜け脚質ペアは追追が最弱(TR54.5%残存=他力2車)・ドメイン整合だがTESTのn小。**下位断絶あり→ROI低下**(TR167/TE158% < なし247/280%)＝実効車数減も市場が先に織込む（弱い負レバー）。
- PG-2(ランク内): S×拮抗がTR111/TE86%と弱い（弱いskip候補・n小）。PG-1b: 形はtop3_sumレバーに概ね包含。PG-4: ガールズ除外で主要セル不変。

## 実験2: 本命優位度のモデルvs市場ギャップ

市場P(top3)は trio全盤面から逆算: q_i=Σ(1/odds)、P_mkt=3q/Σq。D_gap=gap12_model−gap12_mkt。

- **D_gapの素性**: corr(top3_sum)=-0.185（独立）・corr(gap12)=+0.710（モデル側ギャップ主導）・corr(gap12, gap12_mkt)=+0.419（モデルと市場の優位度評価は中相関しかない）。モデル1位=市場本命の一致率73.7%。
- **本命バスト率はD_gap単調**: Q1(市場の方が抜けてると見る)19.6% → Q4(モデルの方が抜け)13.8%＝モデルの確信が市場を上回る時バストは少ない（モデル正しい）。
- **D_gap四分位ROI**: Q4がTR795%/TE808%（最大除737/664%）で頑健だがgap12と高相関＝既存S/SSランクと重複大。
- **★本命人物の不一致**（メイン結果）:

| 区分 | TRAIN | TEST |
|---|---|---|
| 一致(73.7%) | 210% (n3365) | 336% (n369) |
| **不一致** | **1168%** [884,1491] 最大除1101% (n1108) | **576%** [365,809] 最大除521% (n116) |

- top3_sum直交(Part4): loose内/外どちらでもD_gap高側が上（1073vs730 / 140vs97）＝方向一貫・CI重複。
- **留保**: 市場本命の同定もROIも**最終オッズ**。①朝オッズでの本命同定が直前と一致するか（ドリフト）②既存ガミ3段階(≥5倍)との重複度、が本番化前の必須ゲート。
