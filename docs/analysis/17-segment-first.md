# 条件先行（segment-first）全面再検討 — 3期間プロトコル（2026-06-12）

> **結論: 条件先行で85セルをスキャンしても、新しい予想方式は見つからない。**
> 3期間（TRAIN/VAL/HOLDOUT）を点推定で通過したのは **A1=「3連複2軸流し3点×最安オッズ≥5倍」系の
> 3セルのみ**で、いずれも現行運用と同型（残り2つはその部分集合）。新規メニュー
> （3連単BOX無条件・ワイドvalue単独・fav_mismatch単独・波乱ゲート単独）は全てVALで脱落。
> **副発見: リーク排除モデルでは既知レバー単独の数字が大幅に縮む** — 過去の派手なROIには
> 本番モデルの週次再学習（評価期間を学習済み）由来の上振れが含まれていた可能性が高い。
> スクリプト: `scripts/exp_segment_first_wt.py`

## 問い（ユーザー）

現行は「予想ありき→条件で絞る」。逆に最初から条件でセグメントを分け、セグメントごとに
予想アプローチを変える方式をフラットに検討。合格基準 = **TRAIN>100% & VAL>100% &
完全独立HOLDOUT>100%** のみ。

## プロトコル（事前登録・本検証の新規性）

1. **モデルのリーク排除**: 本番 `lgbm_wt` は週次再学習で評価期間を学習済み → **TRAIN期間
   （2023-07〜2025-06）のみで新規学習したLGBM**を全期間の予測に使用。過去の全バックテストと
   異なり、HOLDOUTはモデルにとって真に未知。
2. 期間: TRAIN 2023-07〜2025-06 (3,948R) / VAL 2025-07〜2026-02 (1,097R・セル選別はここまで) /
   **HOLDOUT 2026-03〜2026-06-12 (522R・生存セルのみ最後に1回評価)**
3. セル = セグメント17 × アプローチ5 = 85（多重比較は開示の上、2段ゲートで制御）:
   - セグメント: ALL / 車立て(4,5,6) / 時間帯(〜10・10-16・16-20・20〜) / グレード(A,S,L級) /
     開催日次(1,2,3+) / ライン数(≤2,3,≥4)
   - アプローチ: A1 trio3点×最安≥5倍 / A2 3連単BOX6点無条件 / A3 trio×top3_sum Q1 /
     A4 ワイド1点≥2.5倍 / A5 trio×fav_mismatch（市場本命≠モデル1位）
   - ※モデルの形状フィルタ（gap12/tier）は意図的に外し「条件＝レース属性」のみで分割
4. 選別 = TRAIN>100% & VAL>100% & VAL n≥30 & VAL最大払戻除去>100% → 生存7セル → HOLDOUT一発。

## 結果

### HOLDOUT一発評価（生存7セル）

| セグメント | アプローチ | HOLDOUT | 判定 |
|---|---|---|---|
| ALL | A1 trio最安≥5 | 48R **152%** 的中35% [73,262] 除116% | ✅（点推定で3期間通過） |
| day:3+ | A1 | 26R 182% [63,372] 除115% | ✅（ALLの部分集合） |
| grade:A級 | A1 | 32R 162% [53,319] 除107% | ✅（同上） |
| 時間10-16 | A1 | 16R 318% | 小標本 |
| lines≤2 | A1 | 12R 251% | 小標本 |
| 時間10-16 | A3 波乱Q1 | 48R 119% 除99% | △ 除最大<100 |
| day:3+ | A5 市場不一致 | 50R **73%** | ✗ HOLDOUTで死亡 |

- 合格は**全てA1系**。A1-ALL は TRAIN 185% / VAL 134%(除113%) / HOLDOUT 152%(除116%) と
  3期間を点推定で通過。ただし **HOLDOUT CIは[73%,262%]と100%を跨ぐ**（48Rの限界）＋最終オッズ上限値。
- day:3+×A5 がHOLDOUTで73%に死亡＝**VAL通過でも偶然の生存が混入する**ことの実例
  （85セル走査の多重比較が現実に牙を剥いた。HOLDOUT段の存在意義）。

### 新規メニュー・セグメント分割の学び

- **A2 3連単BOX6無条件**: 全セグメントでTRAIN 78-101%/VAL 59-119%。無条件では成立しない
  （doc10のBOX6はSS形状＋選別込みの話で、構造単体にエッジは無い）。
- **A4 ワイドvalue単独**: ALLでVAL 104%だが除最大95%で脱落（doc12の評価と整合・live継続が正）。
- **A5 fav_mismatch単独 / A3 波乱Q1単独**: VAL 96% / 91%。**リーク無しモデル＋無条件適用では
  単独レバーとして不成立**。既存レバーは「組み合わせの中の選別軸」としてのみ意味を持つ。
- セグメントの質感: エッジは **5車立て**(A1 TRAIN 355%)・**A級**・**後半日程**に濃く、
  **6車立てのA1はTRAIN 92%/VAL 84%と負け**。ただしこれらは事後スライスで、独立な新エッジではない
  （n=5×A1のVALは14Rで判定不能）。ミッドナイト(20時〜)・ガールズ(L級)に特異なエッジは無し。

### 最重要の副発見: モデルリークの示唆

同じ「最安≥5倍」帯でも、本番モデル（週次再学習）では TEST 492%（doc16）、リーク排除モデルでは
HOLDOUT 152%。定義差（tier有無・SS3連単の混入）もあるが、**過去のバックテスト数字には
「モデルが評価期間を学習済み」由来の上振れが乗っている可能性が高い**。実運用ROI（picks_history）
が backtest を大きく下回ってきた一因の候補。今後の検証は本プロトコル（期間限定学習）を標準とすべき。

## 総合判定

- **新方式: 発見なし**。条件先行のセグメント分割は、現行型（オッズ帯選別付きtrio流し）の
  再発見に収束した。これは偶然ではなく、「エッジは選別（どのレースを買うか）にあり、
  買い方の構造や属性分割には無い」という過去の全検証（doc05/09/15/16）と同一の結論。
- ユーザーの合格基準（3期間>100%）を満たす唯一の候補 = **A1: 最安≥5倍×trio 2軸流し3点**
  （形状フィルタ無しでも成立する点は新情報）。ただしCIが100%を跨ぐため、
  **最終確定は前向きlive（picks_history）でのROI実測のみ**。
- 統計的に確定した新エッジは無い、が誠実な答え。

## 再開条件 / やらないこと

- セグメント×アプローチの組み合わせ変種の追加走査 = 多重比較ノイズ。行わない。
- 5車立て集中・形状フィルタ撤廃（gap12なしA1）は、live実測の蓄積で
  「A1-ALLのlive ROI>100%」が見えた場合のみ次の検証候補とする。
- 期間限定学習モデルによる既存レバー（top3_sum・fav_mismatch・mid-odds）の再採点は
  実施価値あり（リーク上振れの定量化）。
