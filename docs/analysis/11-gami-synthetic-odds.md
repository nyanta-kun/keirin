# ガミ判定: 最安レグ vs 合成オッズ の検証（2026-06-09）

> ユーザーの問い:「低オッズを含む買い目をBへ落としているが、**合成オッズ**で判断した方が良いか」。
> スクリプト=`scripts/exp_gami_synthetic_wt.py`。model=lgbm_wt_eval(OOS)・払戻=最終オッズ=**上限値**・TR 4143R/TE 419R。
> 合成オッズ = `1/Σ(1/oᵢ)`（3点束全体の実効オッズ・数学的に常に最安レグ以下）。

## 結論: **合成オッズに変える必要なし（現行=最安レグとほぼ同一・改善は誤差内）**

### A. 両者はほぼ同じ信号
- **相関(最安レグ, 合成) = 0.981**。合成 ≤ 最安レグ 成立率100%。
- 現行バケット内の合成中央値: 見送り<3→1.23 / B[3,5)→1.94 / 推奨≥5→7.48（合成でも明瞭に分離）。

### B. ROI判別力は互角（四分位層別・OOS）
| 帯 | 最安レグ ROI | 合成オッズ ROI |
|---|---|---|
| Q1_安 | 91% | 93% |
| Q2 | 90% | 80% |
| Q3 | 100% | 109% |
| Q4_高 | **692%** | **660%** |

両者とも「利益は最上位四分位に集中・以下は収支トントン」で同形。最上位は最安レグがわずかに上。

### C. 決定的: 両者が食い違うレースがほぼ存在しない
- **現行推奨(最安≥5)のうち合成が低い＝降格候補 = 0件**。構造的に、最安≥5なら3レグ全て≥5→Σ(1/o)≤0.6→合成≥1.67（実中央7.48）。両者は逆転しない。
- 現行見送り(最安<3)で合成≥中央値=過剰スキップ候補は **6件のみ・ROI 28%**＝スキップは正しい。
- **唯一の差＝Bランク帯(最安3-5倍)の内部**: 合成≥中央値 **130%**[76,196]・最大除106% vs 合成<中央値 **69%**[44,95]・最大除64%。合成はB内部をやや良く分けるが**限界的**（CI下限<100）。

### D. 閾値スイープ（推奨=合成≥thr vs 現行=最安≥5）
| 方式 | TE_ROI(R) | TE最大除 |
|---|---|---|
| [現行]最安≥5 | 556%(146) | 471% |
| 合成≥2.5 | 534%(158) | 455% |
| 合成≥3.0 | 595%(135) | 503% |
| 合成≥5.0 | 785%(95) | 655% |

`合成≥3.0` は現行と同程度の購入R(135 vs 146)で ROI 595 vs 556＝**改善は誤差内**（CI大幅重複）。

## なぜ一致するか（構造的説明）
現行の2軸流し3点は **pivot1・pivot2を共有し3着だけ異なる**ため、最安レグ（最も来やすい3着の組）が Σ(1/oᵢ) を支配し、**合成オッズ ≈ 最安レグの単調変換**になる。よって「1点だけ安いが他は高い→不当にBへ落ちている」という懸念はデータ上**起きていない**＝安いレグを含む束は実際に束全体も割安(鉄板)で、B/見送りが正しい判定。

## 推奨
1. **現行の最安レグ3段階は据え置き**（合成への置換はROI改善が誤差・実装/説明の複雑化に見合わない）。
2. 合成オッズの唯一の使い所＝**Bランクの精緻化タグ**: B帯のうち合成≥中央値(~1.84)は+EV寄り(130%)・合成<は鉄板寄り(69%)。Bランク表示に合成オッズを併記すれば各自判断の材料になる。ただし合成≈市場の束的中確率＝既存 `top3_sum`(波乱ゲート)タグと思想重複。
3. 注意: 全て最終オッズ上限値。朝オッズドリフト実測(#2/#3共通)の結論が出れば再評価。

## 補足: 束が多様化すると差が開く可能性
合成≈最安となるのは**3点が2軸を共有する**現構造ゆえ。SS三連単1-2着BOX(6点・`docs/analysis/10`)のように束が多様化すると最安レグと合成が乖離し得る→BOX運用が定着したら合成オッズ判定を再検証する価値あり（現時点では不要）。
