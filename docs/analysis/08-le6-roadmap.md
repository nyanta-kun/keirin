## 確立済みレバー（検証済み・採否判断のための既知事実）

### A. 波乱選別＝最大の収益源（`docs/analysis/01-03`・`exp_upset_gate_wt.py`）
- **本命が堅いレースは低ROI・本命が割れた波乱余地レースが高ROI**（3タスク分析が同一結論に収束）。
- 頑健シグナル=`top3_sum` の loose四分位。Q1_loose 1136%(OOS) vs Q4_chalk 107%。単調・train/test一致。
- AI印(prediction_mark)・セクターは**ROIを下げる**（除去で改善）。**AUC↑≠ROI↑**。

### B. 中間オッズ集中（`docs/analysis/06-mid-odds.md`・`exp_midodds_wt.py`）★再現あり・**運用未反映**
- 2軸流し全点のうち三連複オッズ ∈ [20,80]倍に集中 → TRAIN 116% / TEST 115%（的中4%）。[10,80]倍 → 100/104%（的中9%）。
- **的中率とのトレードオフ**（高的中＝低オッズ＝ROI<100）。top3_sum Q1_loose の別表現（独立な新エッジではない）。
- **昇格の前提=朝→確定オッズのドリフト実測**（最終オッズは上限値。`snapshot_morning_odds_wt.py`で蓄積中）。

### C. ライン力学は転用不要（`docs/analysis/07-le6-line-dynamics.md`）
- 「別ライン頭軸/道連れ崩壊」は≤6車でも構造再現するが、ROIは**現行2軸流しが上回る**（収益核Q1で現行111% vs ライン版99%）。
- ライン情報は既に `line_group`/`n_senko` 等でモデル内包済 → 明示ルール化は的中率を落とすだけ。**現行戦略維持**。

### D. ガミ3段階・Bランク（`analyze_gami_*_wt.py`）
- <3倍点を含むレースは集団で収支ゼロの鉄板 → レース単位振り分けがROI・総損益とも上。
- ~~**Bランク(3〜5倍)は +EV(~116%)**~~ → **2026-06-12更新（`docs/analysis/16-b-lower-cutoff.md`）**: 116%はTRAIN期間の数字。最新OOS(2026-03〜06-11・103営業日)では **86% [59,121] と非再現**。B本採用を正当化する証拠なし＝「各自判断・推奨合計に含めず」を維持。
- **B閾値緩和×買い目足切りは全セル不可**（同doc・2026-06-12）: 見送り帯/B帯/合成帯×足切り(≥2〜≥10/[10,80])のどの組合せもTEST<100%。TRAINだけ有望なセル多数=多重比較の罠。足切りはTRAIN↑TEST↓の典型的過学習挙動。

### E. 副産物（7+検証から ≤6車へ転用候補・未着手）
- **較正済み確率(isotonic)＋Plackett-Luce同時確率**の部品（`exp_7plus_value_wt.py`で実装）→ ≤6車の**Kellyステーク傾斜・ガミ判定精度**に転用余地。
- `r1_prob`(モデル自己信頼度)=バストの較正済み予測子 → 信頼度ゲート転用余地。

## 次に検証する問い（ユーザー目標: 的中率↑・ROI↑・購入レース↑）

優先度は「実運用で効くか」で判断。**規律=★再現はTRAIN(大標本)とOOS TEST両方>100%・bootstrap CI・最大払戻除去・朝オッズドリフト考慮**（`roi_robustness_wt.py`）。

1. **【量↑】Bランクの本採用判断** — ~~+EV(~116%)のBを推奨合計に含めるか~~ **2026-06-12: 否定方向の証拠**（`docs/analysis/16`・OOSでB帯86%非再現・閾値緩和×足切りも全滅）。live実測(`picks_history`)がOOSと矛盾する強い黒字を示さない限り現状維持（各自判断・合計に含めず）。
2. **【ROI↑】中間オッズ帯フィルタの昇格** — [10,80]/[20,80]倍を `wave-picks-wt` のオプション実装→現行フル3点と並行ライブ比較。**朝オッズドリフト実測が前提**（数日蓄積後 `snapshot_morning_odds_wt.py --report`）。
3. **【的中率↑/ROI↑】波乱ゲートの本番化** — `top3_sum` Q1_loose をcron既定にするか・ステーク傾斜にするか。`upset_tier`×`picks_history`の帯別live ROIを数週間蓄積後に判断。
4. ~~**【ROI↑】較正確率→Kellyステーク傾斜**~~ — **検証済・DEFER（`docs/analysis/09-kelly-stake.md`）**。per-comboのvalueエッジは≤6車でも無い(市場効率・7+と同型／較正P≈市場示唆)。成長率の主役は **top3_sum選別**(tilt g=0.30 > kelly単体0.21・H-B確定)。`kelly×tilt` のみ点推定で上回る(g=0.44/ROI1599%)が CI重複・156R・上限値で未確立。本番化せず #2/#3(ドリフト実測ゲート)後に再評価。
5. **【精度↑】新race-levelシグナル探索** — `n_senko`採用済・`style_enc`復活済。展開(先行争い)次元で更なる独立シグナルがあるか（venue/選手IDの過学習は回避）。**展開シナリオ条件付きミクスチャは検証済・撤退**（`docs/analysis/15-scenario-mixture.md`・2026-06-11）: 展開4クラス(逃/捲/番手差し/リーダー差し)は事前予測可能(TEST acc40% vs base34%)だが、Σ P(s)·PL_s の組み合わせ確率改善は有意でなく(ランダム分割コントロールと差なし=容量効果)、市場示唆が全モデルアームに有意優位(Δ-logP +0.16)。per-combo valueも#4と同型の壁。
6. **【前提】実運用ROIの実測と上限値補正** — backtestは最終オッズ上限値。`picks_history(route='wt')`の蓄積で実ROIを測り、上限値からの下振れ率を確定（ks実績は修正後1週間49%だった点に留意）。
7. ~~**【ROI↑/候補】SS層の三連単を 1-2着BOX に拡張**~~ — **2026-06-15 クローズ（`docs/analysis/33-axis-accuracy-trifecta.md`・S2 tri_box）**: doc10 の「★頑健 >100%」は3バイアス込み。リーク無し検証で S2（pred1/pred2 1-2着BOX→thirds 6点）が VAL **32.5%**・HOLD **39.8%** と明確に不成立。三連単への切り替えはいかなる帯でも効果なし。

8. **【新情報源】money-flow（朝→直前オッズ変動）の検証** — **標本蓄積待ち・約9ヶ月**（`docs/analysis/23-moneyflow-initial.md`・G04）。`scripts/exp_moneyflow_wt.py` で3事前登録セル（a:モデル上位目×短縮/伸長 ROI差 b:C0×短縮ゲート c:fav_mismatch×本命交代）を実装済み。現有データ≤6車30R（≈最小1,624Rの2%）で評価不能。`scripts/snapshot_intraday_odds_wt.py`（G03）でスナップショット蓄積中（cron毎時）。再開=≥1,624R蓄積後に `exp_moneyflow_wt.py --report` を再実行。

## やらないこと（確定）
- **7+の再試行**（買い方・軸・モデル・合成オッズ足切り）= 多重比較ノイズ。再開は公開オッズを超える情報源を得た時のみ。
- **展開シナリオ分割・混合の変種**（クラス数変更・決まり手以外のラベル・混合重みの工夫等）= 多重比較ノイズ（`docs/analysis/15`）。再開は P(展開) が市場の織り込みを超える外部情報源（並び実測・直前情報・money-flow）を得た時のみ。
- **ガミ帯の閾値緩和（見送り帯/B帯の取り込み）と買い目足切りの組み合わせ変種** = 全セルTEST<100%で確定（`docs/analysis/16`）。再開は朝→確定オッズドリフト実測に基づく「朝オッズ基準の帯再分類」の再評価のみ。
- **風（気象）特徴の追加** = Phase1 不通過・無情報で確定（`docs/analysis/24-wind-feature.md`・G06）。VAL/HOLD AUC差 ±0.0003 と閾値 ±0.001 未満。wtのline_*等が展開経由で気象を実質織込済と解釈。
- **高配当10点パターンの変種** = `docs/analysis/21-highpay10-leakfree.md`（G07ゲート元情報）でリーク無し全36セル不通過。変種は多重比較ノイズ。再開はmoney-flow蓄積後のゲート再判定のみ。
- venue/選手IDをそのまま特徴化（過学習）。明示的ライン軸置換（C）。AI印・セクターの買い目反映（A）。
- **ライン構造ベース買い目設計（軸シフト）の変種** = 全3戦略全期間ROI<100%で確定（`docs/analysis/32`）。特徴量追加も軸シフトも公開情報の範囲内では市場効率の壁を突破できない。再開はmoney-flow等の公開オッズを超える情報源のみ。
- **三連単・1-2着BOXへの切り替え変種** = リーク無しで S2(tri_box) VAL 32.5%/HOLD 39.8% の不成立確定（`docs/analysis/33`）。いかなる帯でも三連複2軸流しを上回らない。
- **fav_mismatch 単独戦略** = リーク無し初回検証（`docs/analysis/34`）で HOLD 23.1%。1168%/576%（doc13）は3バイアスの産物（6〜25倍過大評価）。live実測が採否の唯一根拠（現在蓄積中）。
- **他式別オッズ（二連単・ワイド・二連複）を特徴量として追加** = Phase1 AUC+0.022 で通過するが Phase2 不通過（VAL 悪化・n 半減）。市場の5式別は統合されており裁定不可（`docs/analysis/35`）。変種は多重比較ノイズ。
- **グレード別モデル（全体）** = Phase1 AUC低下・Phase2 VAL 97.3%（閾値 100% 未達）（`docs/analysis/35`）。S级のみは VAL 107%★ だが HOLD 88%・15R。100R live 蓄積後に再評価。
- **venue × grade 限定 rolling WR** = Phase1 不通過（AUC +0.0001）。venue_wr + player_class_enc で既に代替情報を保有（`docs/analysis/37`）。
- **ライン連携コヒージョン（partner_top3_rate・leader_win_rate）** = Phase1 不通過（AUC +0.0001）。ペア固有の共走が少なく疎すぎる。line_frac/is_line_leader が既に内包（`docs/analysis/38`）。
- **朝→夕方 intraday drift（doc36）** = 方法論確立・C0対象 4R/5日で統計的評価不可（≈12ヶ月待ち）。セル c 方向一致率 100%（15脚）は参考値。overnight drift（前日夕→当日朝）は現在収集未実装。
